Source Code

This is a reconstructed version of Luminary 130, the initial AGC program released in November 1969 for the Apollo 13 LM. Three more releases followed before the flown revision was finally manufactured in February 1970. This reconstruction was created from a previous transcription of the original listing of Luminary 131, combined with Luminary memo #129 which describes the small difference between the Luminary 130 and 131. The reconstructed source code was verified by matching memory-bank checksums to those listed for Luminary 130 in drawing 2021152G. In this source code, program comments from the original (Luminary 131) code are prefixed with '#', while those added later by the Virtual AGC project are prefixed with '##'.

054037,000027:                                                                                                  #  PROGRAM DESCRIPTION                           DATE  20 DECEMBER 1967
054038,000028:                                                                                                  #  PROGRAM NAME - SELF-CHECK                     LOG SECTION  AGC BLOCK TWO SELF-CHECK
054039,000029:                                                                                                  #  MOD NO - 1                                    ASSEMBLY SUBROUTINE UTILITYM REV 25
054040,000030:                                                                                                  #  MOD BY - GAUNTT
054041,000031: 
054042,000032:                                                                                                  #  FUNCTIONAL DESCRIPTION
054043,000033:                                                                                                  #        PROGRAM HAS TWO MAIN PARTS.  THE FIRST IS SELF-CHECK WHICH RUNS AS A ZERO PRIORITY JOB WITH NO CORE SET, AS
054044,000034:                                                                                                  #        PART OF THE BACK-UP IDLE LOOP.  THE SECOND IS SHOW-BANKSUM WHICH RUNS AS A REGULAR EXECUTIVE JOB WITH ITS OWN
054045,000035:                                                                                                  #        STARTING VERB.
054046,000036: 
054047,000037:                                                                                                  #        THE PURPOSE OF SELF-CHECK IS TO CHECK OUT VARIOUS PARTS OF THE COMPUTER AS OUTLINED BELOW IN THE OPTIONS.
054048,000038: 
054049,000039:                                                                                                  #        THE PURPOSE OF SHOW-BANKSUM IS TO DISPLAY THE SUM OF EACH BANK, ONE AT A TIME.
054050,000040: 
054051,000041:                                                                                                  #        IN ALL THERE ARE 7 POSSIBLE OPTIONS IN THIS BLOCK II VERSION OF SELF-CHECK.  MORE DETAIL DESCRIPTION MAY BE
054052,000042:                                                                                                  #        FOUND IN E-2065 BLOCK II AGC SELF-CHECK AND SHOW BANKSUM BY EDWIN D. SMALLY DECEMBER 1966, AND ADDENDA 2 AND 3.
054053,000043: 
054054,000044:                                                                                                  #        THE DIFFERENT OPTIONS ARE CONTROLLED BY PUTTING DIFFERENT NUMBERS IN THE SMODE REGISTER (NOUN 27).  BELOW IS
054055,000045:                                                                                                  #        A DESCRIPTION OF WHAT PARTS OF THE COMPUTER THAT ARE CHECKED BY THE OPTIONS, AND THE CORRESPONDING NUMBER, IN
054056,000046:                                                                                                  #        OCTAL, TO LOAD INTO SMODE.
054057,000047:                                                                                                  #                +-4             ERASABLE MEMORY
054058,000048:                                                                                                  #                +-5             FIXED MEMORY
054059,000049:                                                                                                  #                +-1,2,3,6,7,10  EVERYTHING IN OPTIONS 4 AND 5.
054060,000050:                                                                                                  #                -0              SAME AS +-10 UNTIL AN ERROR IS DETECTED.
054061,000051:                                                                                                  #                +0              NO CHECK, PUTS COMPUTER INTO THE BACKUP IDLE LOOP.
054062,000052: 
054063,000053:                                                                                                  #  WARNINGS
054064,000054:                                                                                                  #        USE OF E MEMORY RESERVED FOR SELF-CHECK (EVEN IN IDLE LOOP) AS TEMP STORAGE BY OTHER PROGRAMS IS DANGEROUS.
054065,000055:                                                                                                  #        SMODE SET GREATER THAN OCT 10 PUTS COMPUTER INTO BACKUP IDLE LOOP.
054066,000056: 
054067,000057:                                                                                                  #  CALLING SEQUENCE
054068,000058:                                                                                                  #        TO CALL SELF-CHECK KEY IN
054069,000059:                                                                                                  #                V 21 N 27 E     OPTION NUMBER E
054070,000060:                                                                                                  #        TO CALL SHOW-BANKSUM KEY IN
054071,000061:                                                                                                  #                V 91 E          DISPLAYS FIRST BANK
054072,000062:                                                                                                  #                V 33 E          PROCEED, DISPLAYS NEXT BANK
054073,000063: 
054074,000064:                                                                                                  #  EXIT MODES, NORMAL AND ALARM
054075,000065:                                                                                                  #        SELF-CHECK NORMALLY CONTINUES INDEFINITELY UNLESS THERE IS AN ERROR DETECTED.  IF SO + OPTION NUMBERS PUT
054076,000066:                                                                                                  #        COMPUTER INTO BACKUP IDLE LOOP, - OPTION NUMBERS RESTART THE OPTION.
054077,000067: 
054078,000068:                                                                                                  #        THE -0 OPTION PROCEEDS FROM THE LINE FOLLOWING THE LINE WHERE THE ERROR WAS DETECTED.
054079,000069: 
054080,000070:                                                                                                  #        SHOW-BANKSUM PROCEEDS UNTIL A TERMINATE IS KEYED IN (V 34 E).  THE COMPUTER IS PUT INTO THE BACKUP IDLE LOOP
054081,000071: 
054082,000072:                                                                                                  #  OUTPUT

054084,000074:                                                                                                  #        SELF-CHECK UPON DETECTING AN ERROR LOADS THE SELF-CHECK ALARM CONSTANT (01102) INTO THE FAILREG SET AND
054085,000075:                                                                                                  #        TURNS ON THE ALARM LIGHT.  THE OPERATOR MAY THEN DISPLAY THE THREE FAILREGS BY KEYING IN V 05 N 09 E.  FOR FURTHER
054086,000076:                                                                                                  #        INFORMATION HE MAY KEY IN V 05 N 08 E, THE DSKY DISPLAY IN R1 WILL BE ADDRESS+1 OF WHERE THE ERROR WAS DETECTED,
054087,000077:                                                                                                  #        IN R2 THE BBCON OF SELF-CHECK, AND IN R3 THE TOTAL NUMBER OF ERRORS DETECTED BY SELF-CHECK SINCE THE LAST MAN
054088,000078:                                                                                                  #        INITIATED FRESH START (SLAP1).
054089,000079: 
054090,000080:                                                                                                  #        SHOW-BANKSUM STARTING WITH BANK 0 DISPLAYS IN R1 THE BANK SUM (A +-NUMBER EQUAL TO THE BANK NUMBER), IN R2
054091,000081:                                                                                                  #        THE BANK NUMBER, AND IN R3 THE BUGGER WORD.
054092,000082: 
054093,000083:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
054094,000084:                                                                                                  #        ACCOMPLISHED BY FRESH START
054095,000085:                                                                                                  #                SMODE SET TO +0
054096,000086: 
054097,000087:                                                                                                  #  DEBRIS
054098,000088:                                                                                                  #        ALL EXITS FROM THE CHECK OF ERASABLE (ERASCHK) RESTORE ORIGINAL CONTENTS TO REGISTERS UNDER CHECK.
054099,000089:                                                                                                  #        EXCEPTION IS A RESTART.  RESTART THAT OCCURS DURING ERASCHK RESTORES ERASABLE, UNLESS THERE IS EVIDENCE TO DOUBT
054100,000090:                                                                                                  #        E MEMORY, IN WHICH CASE PROGRAM THEN DOES A FRESH START (DOFSTART).
