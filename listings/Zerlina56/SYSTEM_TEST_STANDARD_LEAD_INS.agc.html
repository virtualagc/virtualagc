Source Code

These source-code files were transcribed from a printout in Don Eyles's personal collection, scanned by archive.org, and financially sponsored by Linden Sims. A team of volunteers performed the transcription and proof-reading. The scanned page images are available at the Virtual AGC Project website, as well as higher-quality (but much larger) images at the Virtual AGC Project's collection in the Internet Archive. Report any problems by creating "issues" at the Virtual AGC Project's GitHub Repository.

016673,000033:                                                                                                  #  E/BKCALL   FOR DOING BANKCALLS FROM AND RETURNING TO ERASABLE.
016674,000034: 
016675,000035:                                                                                                  #           THIS ROUTINE IS CALLABLE FROM ERASABLE OR FIXED.  LIKE BANKCALL, HOWEVER, SWITCHING BETWEEN S3 AND S4
016676,000036:                                                                                                  #  IS NOT POSSIBLE.
016677,000037: 
016678,000038:                                                                                                  #           THE CALLING SEQUENCE IS:
016679,000039: 
016680,000040:                                                                                                  #                                                   TC     BANKCALL
016681,000041:                                                                                                  #                                                   CADR   E/BKCALL
016682,000042:                                                                                                  #                                                   CADR   ROUTINE         WHERE YOU WANT TO GO IN FIXED.
016683,000043:                                                                                                  #                                                   RETURN HERE FROM DISPLAY TERMINATE, BAD STALL OR TC Q.
016684,000044:                                                                                                  #                                                   RETURN HERE FROM DISPLAY PROCEED OR GOOD RETURN FROM STALL.
016685,000045:                                                                                                  #                                                   RETURN HERE FROM DISPLAY ENTER OR RECYCLE.
016686,000046: 
016687,000047:                                                                                                  #           THIS ROUTINE REQUIRES TWO ERASABLES (EBUF2, +1) IN UNSWITCHED WHICH ARE UNSHARED BY INTERRUPTS AND
016688,000048:                                                                                                  #  OTHER EMEMORY PROGRAMS.
016689,000049: 
016690,000050:                                                                                                  #           A + L ARE PRESERVED THROUGH BANKCALL AND E/BKCALL.

016694,000054: 04,2572           25166                           INCR     EBUF2                                 #  RETURN +1 BECAUSE DOUBLE CADR.

016696,000056: 04,2574           75012                           MASK     LOW10                                 #  GET CURRENT EBANK.  (SBANK SOMEDAY)

016698,000058: 04,2576           51166                           NDX      EBUF2
