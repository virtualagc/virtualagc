Source Code

These source-code files are part of a reconstructed copy of Luminary 173, the second (unflown) release of the Apollo 14 Lunar Module (LM) Apollo Guidance Computer (AGC) software.

The reconstruction began with reconstructed source code for Luminary 178. Changes between revision 173 and 178 were backed out, as described by Luminary memo 167 (revision 1). The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021152N. Note that page numbers in the reconstructed code match those for the baseline log section mentioned in each file's changelog; the page numbers for a real Luminary 173 listing would be different.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

054331,000027:                                                                                                  #  FUNCTIONAL DESCRIPTION
054332,000028: 
054333,000029:                                                                                                  #       PROGRAM HAS TWO MAIN PARTS. THE FIRST IS SELF-CHECK WHICH RUNS AS A ZERO PRIORITY JOB WITH NO CORE SET, AS
054334,000030:                                                                                                  #  PART OF THE BACK-UP IDLE LOOP. THE SECOND IS SHOW-BANKSUM WHICH RUNS AS A REGULAR EXECUTIVE JOB WITH ITS OWN
054335,000031:                                                                                                  #  STARTING VERB.
054336,000032:                                                                                                  #       THE PURPOSE OF SELF-CHECK IS TO CHECK OUT VARIOUS PARTS OF THE COMPUTER AS OUTLINED BELOW IN THE OPTIONS.
054337,000033:                                                                                                  #       THE PURPOSE OF SHOW-BANKSUM IS TO DISPLAY THE SUM OF EACH BANK , ONE AT A TIME.
054338,000034:                                                                                                  #       IN ALL THERE ARE  7 POSSIBLE OPTIONS IN THIS BLOCK II VERSION OF SELF-CHECK. MORE DETAIL DESCRIPTION MAY BE
054339,000035:                                                                                                  #  FOUND IN E-2065 BLOCK II AGC SELF-CHECK AND SHOW BANKSUM BY EDWIN D. SMALLY DECEMBER 1966, AND ADDENDA 2 AND 3.
054340,000036:                                                                                                  #       THE DIFFERENT OPTIONS ARE CONTROLLED BY PUTTING DIFFERENT NUMBERS IN THE SMODE REGISTER (NOUN 27). BELOW IS
054341,000037:                                                                                                  #  A DESCRIPTION OF WHAT PARTS OF THE COMPUTER THAT ARE CHECKED BY THE OPTIONS, AND THE CORRESPONDING NUMBER, IN
054342,000038:                                                                                                  #  OCTAL, TO LOAD INTO SMODE.
054343,000039:                                                                                                  #  +-4   ERASABLE MEMORY
054344,000040:                                                                                                  #  +-5   FIXED MEMORY
054345,000041:                                                                                                  #  +-1,2,3,6,7,10   EVERYTHING IN OPTIONS 4 AND 5.
054346,000042:                                                                                                  #  -0    SAME AS +-10 UNTIL AN ERROR IS DETECTED.
054347,000043:                                                                                                  #  +0    NO CHECK, PUTS COMPUTER INTO THE BACKUP IDLE LOOP.
054348,000044: 
054349,000045: 
054350,000046:                                                                                                  #  WARNINGS
054351,000047: 
054352,000048:                                                                                                  #       USE OF E MEMORY RESERVED FOR SELF-CHECK (EVEN IN IDLE LOOP) AS TEMP STORAGE BY OTHER PROGRAMS IS DANGEROUS.
054353,000049:                                                                                                  #       SMODE SET GREATER THAN OCT 10 PUTS COMPUTER INTO BACKUP IDLE LOOP.
054354,000050: 
054355,000051: 
054356,000052:                                                                                                  #  CALLING SEQUENCE
054357,000053: 
054358,000054:                                                                                                  #       TO CALL SELF-CHECK KEY IN
054359,000055:                                                                                                  #            V 21 N 27 E  OPTION NUMBER E
054360,000056:                                                                                                  #       TO CALL SHOW-BANKSUM KEY IN
054361,000057:                                                                                                  #            V 91 E         DISPLAYS FIRST BANK
054362,000058:                                                                                                  #            V 33 E         PROCEED, DISPLAYS NEXT BANK
054363,000059: 
054364,000060: 
054365,000061:                                                                                                  #  EXIT MODES, NORMAL AND ALARM
054366,000062: 
054367,000063:                                                                                                  #       SELF-CHECK NORMALLY CONTINUES INDEFINITELY UNLESS THERE IS AN ERROR DETECTED. IF SO + OPTION NUMBERS PUT
054368,000064:                                                                                                  #  COMPUTER INTO BACKUP IDLE LOOP, - OPTION NUMBERS RESTART THE OPTION.
054369,000065:                                                                                                  #       THE -0 OPTION PROCEEDS FROM THE LINE FOLLOWING THE LINE WHERE THE ERROR WAS DETECTED.
054370,000066:                                                                                                  #       SHOW-BANKSUM PROCEEDS UNTIL A TERMINATE IS KEYED IN (V 34 E). THE COMPUTER IS PUT INTO THE BACKUP IDLE LOOP
054371,000067:

054376,000072:                                                                                                  #       SELF-CHECK UPON DETECTING AN ERROR LOADS THE SELF-CHECK ALARM CONSTANT (01102) INTO THE FAILREG SET AND
054377,000073:                                                                                                  #  TURNS ON THE ALARM LIGHT. THE OPERATOR MAY THEN DISPLAY THE THREE FAILREGS BY KEYING IN V 05 N 09 E. FOR FURTHER
054378,000074:                                                                                                  #  INFORMATION HE MAY KEY IN V 05 N 08 E, THE DSKY DISPLAY IN R1 WILL BE ADDRESS+1 OF WHERE THE ERROR WAS DETECTED,
054379,000075:                                                                                                  #  IN R2 THE BBCON OF SELF-CHECK, AND IN R3 THE TOTAL NUMBER OF ERRORS DETECTED BY SELF-CHECK SINCE THE LAST MAN
054380,000076:                                                                                                  #  INITIATED FRESH START (SLAP1).
054381,000077:                                                                                                  #       SHOW-BANKSUM STARTING WITH BANK 0 DISPLAYS IN R1 THE BANK SUM (A +-NUMBER EQUAL TO THE BANK NUMBER), IN R2
054382,000078:                                                                                                  #  THE BANK NUMBER, AND IN R3 THE BUGGER WORD.
054383,000079: 
054384,000080: 
054385,000081:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
054386,000082: 
054387,000083:                                                                                                  #       ACCOMPLISHED BY FRESH START
054388,000084:                                                                                                  #            SMODE SET TO +0
054389,000085: 
054390,000086: 
054391,000087:                                                                                                  #  DEBRIS
054392,000088: 
054393,000089:                                                                                                  #       ALL EXITS FROM THE CHECK OF ERASABLE (ERASCHK) RESTORE ORIGINAL CONTENTS TO REGISTERS UNDER CHECK.
054394,000090:                                                                                                  #  EXCEPTION IS A RESTART. RESTART THAT OCCURS DURING ERASCHK RESTORES ERASABLE, UNLESS THERE IS EVIDENCE TO DOUBT
054395,000091:                                                                                                  #  E MEMORY, IN WHICH CASE PROGRAM THEN DOES A FRESH START (DOFSTART).
