These source-code files were obtained by digitally photographing an Artemis 72 (Apollo 15-17 Command Module) program listing from a private collection and then modifying pre-existing Comanche 55 (Apollo 11) source files to incorporate changes. Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS 
	BY NASA 2021114-011 11:40 FEB. 26, 1971
Note that the date is the date of the printout, not the date of the program revision.

059691,000020:                                                                                                  #  PROGRAM NAME -  SELF-CHECK                    LOG SECTION AGC BLOCK TWO SELF-CHECK
059692,000021:                                                                                                  #  MOD NO - 1                                    ASSEMBLY SUBROUTINE UTILITYM REV 25
059693,000022:                                                                                                  #  MOD BY - GAUNTT

059695,000024:                                                                                                  #  FUNCTIONAL DESCRIPTION
059696,000025: 
059697,000026:                                                                                                  #  PROGRAM HAS TWO MAIN PARTS. THE FIRST IS SELF-CHECK WHICH RUNS AS A ZERO PRIORITY JOB WITH NO CORE SET, AS
059698,000027:                                                                                                  #  PART OF THE BACK-UP IDLE LOOP. THE SECOND IS SHOW-BANKSUM WHICH RUNS AS A REGULAR EXECUTIVE JOB WITH ITS OWN
059699,000028:                                                                                                  #  STARTING VERB.
059700,000029:                                                                                                  #  THE PURPOSE OF SELF-CHECK IS TO CHECK OUT VARIOUS PARTS OF THE COMPUTER AS OUTLINED BELOW IN THE OPTIONS.
059701,000030:                                                                                                  #  THE PURPOSE OF SHOW-BANKSUM IS TO DISPLAY THE SUM OF EACH BANK, ONE AT A TIME.
059702,000031:                                                                                                  #  IN ALL THERE ARE  7 POSSIBLE OPTIONS IN THIS BLOCK II VERSION OF SELF-CHECK. MORE DETAIL DESCRIPTION MAY BE
059703,000032:                                                                                                  #  FOUND IN E-2065 BLOCK II AGC SELF-CHECK AND SHOW BANKSUM BY EDWIN D. SMALLY DECEMBER 1966, AND ADDENDA 2 AND 3.
059704,000033:                                                                                                  #  THE DIFFERENT OPTIONS ARE CONTROLLED BY PUTTING DIFFERENT NUMBERS IN THE SMODE REGISTER (NOUN 27). BELOW IS
059705,000034:                                                                                                  #  A DESCRIPTION OF WHAT PARTS OF THE COMPUTER THAT ARE CHECKED BY THE OPTIONS, AND THE CORRESPONDING NUMBER, IN
059706,000035:                                                                                                  #  OCTAL, TO LOAD INTO SMODE.
059707,000036:                                                                                                  #        +-4             ERASABLE MEMORY
059708,000037:                                                                                                  #        +-5             FIXED MEMORY
059709,000038:                                                                                                  #        +-1,2,3,6,7,10  EVERYTHING IN OPTIONS 4 AND 5.
059710,000039:                                                                                                  #        -0              SAME AS +-10 UNTIL AN ERROR IS DETECTED.
059711,000040:                                                                                                  #        +0              NO CHECK, PUTS COMPUTER INTO THE BACKUP IDLE LOOP.
059712,000041: 
059713,000042:                                                                                                  #  WARNINGS
059714,000043: 
059715,000044:                                                                                                  #        USE OF E MEMORY RESERVED FOR SELF-CHECK (EVEN IN IDLE LOOP) AS TEMP STORAGE BY OTHER PROGRAMS IS DANGEROUS.
059716,000045:                                                                                                  #        SMODE SET GREATER THAN OCT 10 PUTS COMPUTER INTO BACKUP IDLE LOOP.
059717,000046: 
059718,000047:                                                                                                  #  CALLING SEQUENCE
059719,000048: 
059720,000049:                                                                                                  #        TO CALL SELF-CHECK KEY IN
059721,000050:                                                                                                  #                V 21 N 27 E     OPTION NUMBER E
059722,000051:                                                                                                  #        TO CALL SHOW-BANKSUM KEY IN
059723,000052:                                                                                                  #                V 91 E          DISPLAYS FIRST BANK
059724,000053:                                                                                                  #                V 33 E          PROCEED, DISPLAYS NEXT BANK
059725,000054: 
059726,000055: 
059727,000056:                                                                                                  #  EXIT MODES, NORMAL AND ALARM
059728,000057: 
059729,000058:                                                                                                  #  SELF-CHECK NORMALLY CONTINUES INDEFINITELY UNLESS THERE IS AN ERROR DETECTED. IF SO + OPTION NUMBERS PUT
059730,000059:                                                                                                  #  COMPUTER INTO BACKUP IDLE LOOP, - OPTION NUMBERS RESTART THE OPTION.
059731,000060:                                                                                                  #  THE -0 OPTION PROCEEDS FROM THE LINE FOLLOWING THE LINE WHERE THE ERROR WAS DETECTED.
059732,000061:                                                                                                  #  SHOW-BANKSUM PROCEEDS UNTIL A TERMINATE IS KEYED IN (V 34 E). THE COMPUTER IS PUT INTO THE BACKUP IDLE LOOP
059733,000062: 
059734,000063: 
059735,000064:                                                                                                  #  OUTPUT

059737,000066:                                                                                                  #  SELF-CHECK UPON DETECTING AN ERROR LOADS THE SELF-CHECK ALARM CONSTANT (01102) INTO THE FAILREG SET AND
059738,000067:                                                                                                  #  TURNS ON THE ALARM LIGHT. THE OPERATOR MAY THEN DISPLAY THE THREE FAILREGS BY KEYING IN V 05 N 09 E. FOR FURTHER
059739,000068:                                                                                                  #  INFORMATION HE MAY KEY IN V 05 N 08 E, THE DSKY DISPLAY IN R1 WILL BE ADDRESS+1 OF WHERE THE ERROR WAS DETECTED,
059740,000069:                                                                                                  #  IN R2 THE BBCON OF SELF-CHECK, AND IN R3 THE TOTAL NUMBER OF ERRORS DETECTED BY SELF-CHECK SINCE THE LAST MAN
059741,000070:                                                                                                  #  INITIATED FRESH START (SLAP1).
059742,000071:                                                                                                  #  SHOW-BANKSUM STARTING WITH BANK 0 DISPLAYS IN R1 THE BANK SUM (A +-NUMBER EQUAL TO THE BANK NUMBER), IN R2
059743,000072:                                                                                                  #  THE BANK NUMBER, AND IN R3 THE BUGGER WORD.
059744,000073: 
059745,000074: 
059746,000075:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
059747,000076: 
059748,000077:                                                                                                  #        ACCOMPLISHED BY FRESH START
059749,000078:                                                                                                  #                SMODE SET TO +0
059750,000079: 
059751,000080:                                                                                                  #  DEBRIS
059752,000081: 
059753,000082:                                                                                                  #        ALL EXITS FROM THE CHECK OF ERASABLE (ERASCHK) RESTORE ORIGINAL CONTENTS TO REGISTERS UNDER CHECK.
059754,000083:                                                                                                  #        EXCEPTION IS A RESTART. RESTART THAT OCCURS DURING ERASCHK RESTORES ERASABLE, UNLESS THERE IS EVIDENCE TO DOUBT
059755,000084:                                                                                                  #        E MEMORY, IN WHICH CASE PROGRAM THEN DOES A FRESH START (DOFSTART).
