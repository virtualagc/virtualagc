These source-code files were obtained by digitally photographing an Artemis 72 (Apollo 15-17 Command Module) program listing from a private collection and then modifying pre-existing Comanche 55 (Apollo 11) source files to incorporate changes. Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS 
	BY NASA 2021114-011 11:40 FEB. 26, 1971
Note that the date is the date of the printout, not the date of the program revision.

013474,000292:                                                                                                  #  SUBROUTINE NAME:  EARTH ROTATOR       (EARROT1 OR EARROT2)            DATE:  15 FEB 67
013475,000293:                                                                                                  #  MOD NO:  N +1                                                         LOG SECTION:  POWERED FLIGHT SUBROS
013476,000294:                                                                                                  #  MOD BY:  ENTRY GROUP (BAIRNSFATHER)
013477,000295: 
013478,000296:                                                                                                  #  FUNCTIONAL DESCRIPTION:  THIS ROUTINE PROJECTS THE INITIAL EARTH TARGET VECTOR   RTINIT   AHEAD THROUGH
013479,000297:                                                                                                  #        THE ESTIMATED TIME OF FLIGHT. INITIAL CALL RESOLVES THE INITIAL TARGET VECTOR   RTINIT   INTO EASTERLY
013480,000298:                                                                                                  #        AND NORMAL COMPONENTS   RTEAST   AND   RTNORM   . INITIAL AND SUBSEQUENT CALLS ROTATE THIS VECTOR
013481,000299:                                                                                                  #        ABOUT THE (FULL) UNIT POLAR AXIS   UNITW   THROUGH THE ANGLE   WIE DTEAROT   TO OBTAIN THE ROTATED
013482,000300:                                                                                                  #        TARGET VECTOR   RT   . ALL VECTORS EXCEPT   UNITW   ARE HALF UNIT.
013483,000301:                                                                                                  #        THE EQUATIONS ARE
013484,000302:                                                                                                  #                -    -        -                      -
013485,000303:                                                                                                  #                RT = RTINIT + RTNORM (COS(WT) - 1) + RTEAST SIN(WT)
013486,000304: 
013487,000305:                                                                                                  #        WHERE   WT = WIE DTEAROT
013488,000306: 
013489,000307:                                                                                                  #                RTINIT = INITIAL TARGET VECTOR
013490,000308:                                                                                                  #                -        -       -
013491,000309:                                                                                                  #                RTEAST = UNITW * RTINIT
013492,000310:                                                                                                  #                -        -        -
013493,000311:                                                                                                  #                RTNORM = RTEAST * UNITW
013494,000312: 
013495,000313:                                                                                                  #        FOR CONTINUOUS UPDATING, ONLY ONE ENTRY TO EARROT1 IS REQUIRED, WITH SUBSEQUENT ENTRIES AT EARROT2.
013496,000314: 
013497,000315:                                                                                                  #  CALLING SEQUENCE:     FIRST CALL                      SUBSEQUENT CALL
