Source Code

These source-code files were obtained by digitally photographing an Artemis 72 (Apollo 15-17 Command Module) program listing from a private collection and then modifying pre-existing Comanche 55 (Apollo 11) source files to incorporate changes. Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS 
	BY NASA 2021114-011 11:40 FEB. 26, 1971
Note that the date is the date of the printout, not the date of the program revision.

047834,000022:                                                                                                  #  PROGRAM NAME- DOWN TELEMETRY PROGRAM
047835,000023:                                                                                                  #  MOD NO.- 0      TO COMPLETELY REWRITE THE DOWN TELEMETRY PROGRAM AND DOWNLINK ERASABLE DUMP PROGRAM FOR THE
047836,000024:                                                                                                  #                  PURPOSE OF SAVING APPROXIMATELY 150 WORDS OF CORE STORAGE.
047837,000025:                                                                                                  #                  THIS CHANGE REQUIRES AN ENTIRELY NEW METHOD OF SPECIFYING DOWNLINK LISTS. REFER TO DOWNLINK
047838,000026:                                                                                                  #                  LISTS LOG SECTION FOR MORE DETAILS. HOWEVER THIS CHANGE WILL NOT AFFECT THE GROUND PROCESSING
047839,000027:                                                                                                  #                  OF DOWN TELEMETRY DATA.
047840,000028:                                                                                                  #  MOD BY-  KILROY, SMITH, DEWITT
047841,000029:                                                                                                  #  MOD BY - DENSMORE - JUNE 1969 - ELIMINATE ERASABLE DUMP COUNT
047842,000030:                                                                                                  #  DATE-    02 OCT 67
047843,000031:                                                                                                  #  AUTHORS- KILROY, SMITH, DEWITT, DEWOLF, FAGIN
047844,000032:                                                                                                  #  LOG SECTION- DOWN-TELEMETRY PROGRAM
047845,000033:                                                                                                  #  FUNCTIONAL DESCRIPTION- THIS ROUTINE IS INITIATED BY TELEMETRY END
047846,000034:                                                                                                  #           PULSE FROM THE DOWNLINK TELEMETRY CONVERTER. THIS PULSE OCCURS
047847,000035:                                                                                                  #           AT 50 TIMES PER SEC (EVERY 20 MS) THEREFORE DODOWNTM IS
047848,000036:                                                                                                  #           EXECUTED AT THESE RATES. THIS ROUTINE SELECTS THE APPROPRIATE
047849,000037:                                                                                                  #           AGC DATA TO BE TRANSMITTED DOWNLINK AND LOADS IT INTO OUTPUT
047850,000038:                                                                                                  #           CHANNELS 34 AND 35. THE INFORMATION IS THEN GATED OUT FROM THE
047851,000039:                                                                                                  #           LGC IN SERIAL FASHION.
047852,000040:                                                                                                  #           THIS PROGRAM IS CODED FOR A 2 SECOND DOWNLIST. SINCE DOWNRUPTS
047853,000041:                                                                                                  #           OCCUR EVERY 20 MS AND 2 AGC COMPUTER WORDS CAN BE PLACED IN
047854,000042:                                                                                                  #           CHANNELS 34 AND 35 DURING EACH DOWNRUPT THE PROGRAM IS CAPABLE
047855,000043:                                                                                                  #           OF SENDING 200 AGC WORDS EVERY 2 SECONDS.
047856,000044:                                                                                                  #  CALLING SEQUENCE- NONE
047857,000045:                                                                                                  #           PROGRAM IS ENTERED VIA TCF DODOWNTM WHICH IS EXECUTED AS A
047858,000046:                                                                                                  #           RESULT OF A DOWNRUPT. CONTROL IS RETURNED VIA TCF RESUME WHICH
047859,000047:                                                                                                  #           IN EFFECT IS A RESUME.
047860,000048:                                                                                                  #  SUBROUTINES CALLED- NONE
047861,000049:                                                                                                  #  NORMAL EXIT MODE- TCF RESUME
047862,000050:                                                                                                  #  ALARM OR ABORT EXIT MODE- NONE
047863,000051:                                                                                                  #  RESTART PROTECTION:
047864,000052:                                                                                                  #           ON A FRESH START AND RESTART THE 'STARTSUB' SUBROUTINE WILL INITIALIZE THE DOWNLIST POINTER (ACTUALLY
047865,000053:                                                                                                  #           DNTMGOTO) TO THE BEGINNING OF THE CURRENT DOWNLIST (I.E. CURRENT CONTENTS OF DNLSTADR). THIS HAS THE
047866,000054:                                                                                                  #           EFFECT OF IGNORING THE REMAINDER OF THE DOWNLIST WHICH THE DOWN-TELEMETRY PROGRAM WAS WORKING ON WHEN
047867,000055:                                                                                                  #           THE RESTART (OR FRESH START) OCCURRED AND RESUME DOWN TELEMETRY FROM THE BEGINNING OF THE CURRENT
047868,000056:                                                                                                  #           DOWNLIST.
047869,000057:                                                                                                  #           ALSO OF INTEREST IS THE FACT THAT ON A RESTART THE AGC WILL ZERO DOWNLINK CHANNELS 13, 34 AND 35.
047870,000058:                                                                                                  #  DOWNLINK LIST SELECTION:
047871,000059:                                                                                                  #           THE APPROPRIATE DOWNLINK LISTS ARE SELECTED BY THE FOLLOWING:
047872,000060:                                                                                                  #           1. FRESH START
047873,000061:                                                                                                  #           2. V37EXXE  WHERE XX = THE MAJOR MODE BEING SELECTED.
047874,000062:                                                                                                  #           3. UPDATE PROGRAM (P27)
047875,000063:                                                                                                  #           4.   NON-V37 SELECTABLE TYPE PROGRAMS (E.G. AGS INITIALIZATION (SUNDANCE, LUMINARY) AND P61-P62
047876,000064:                                                                                                  #                TRANSITION (COLOSSUS) ETC.).
047877,000065:                                                                                                  #  DOWNLINK LIST RULES AND LIMITATIONS:
047878,000066:                                                                                                  #           READ SECTION(S) WHICH FOLLOW 'DEBRIS' WRITEUP.
047879,000067:                                                                                                  #  OUTPUT-  EVERY 2 SECONDS 100 DOUBLE PRECISION WORDS (I.E. 200 LGC
047880,000068:                                                                                                  #           COMPUTER WORDS) ARE TRANSMITTED VIA DOWNLINK.
047881,000069:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED- NONE
047882,000070:                                                                                                  #           'DNTMGOTO' AND 'DNLSTADR' ARE INITIALIZED BY THE FRESH START PROGRAM.
047883,000071:                                                                                                  #  DEBRIS (ERASABLE LOCATIONS DESTROYED BY THIS PROGRAM)-

047891,000079:                                                                                                  #  NOTES REGARDING DOWNLINK LISTS ASSOCIATED WITH THIS PROGRAM:
047892,000080:                                                                                                  #  1. DOWNLISTS. - DOWNLISTS MUST BE COMPILED IN THE SAME BANK AS THE
047893,000081:                                                                                                  #     DOWN-TELEMETRY PROGRAM. THIS IS DONE FOR EASE OF CODING, FASTER
047894,000082:                                                                                                  #     EXECUTION.
047895,000083:                                                                                                  #  2. EACH DOWNLINK LIST CONSISTS OF A CONTROL LIST AND A NUMBER OF
047896,000084:                                                                                                  #     SUBLISTS.
047897,000085:                                                                                                  #  3. A SUBLIST REFERS TO A SNAPSHOT OR DATA COMMON TO THE SAME OR OTHER
047898,000086:                                                                                                  #     DOWNLINK LISTS. ANY SUBLIST CONTAINING COMMON DATA NEEDS TO BE
047899,000087:                                                                                                  #     CODED ONLY ONCE FOR THE APPLICABLE DOWNLINK LISTS.
047900,000088:                                                                                                  #  4. SNAPSHOT SUBLISTS REFER SPECIFICALLY TO HOMOGENOUS DATA WHICH MUST BE
047901,000089:                                                                                                  #     SAVED IN A BUFFER DURING ONE DOWNRUPT.
047902,000090:                                                                                                  #  5. THE 1DNADR FOR THE 1ST WORD OF SNAPSHOT DATA IS FOUND AT THE END
047903,000091:                                                                                                  #     OF EACH SNAPSHOT SUBLIST, SINCE THE PROGRAM CODING SENDS THIS DP WORD
047904,000092:                                                                                                  #     IMMEDIATELY AFTER STORING THE OTHERS IN THE SNAPSHOT BUFFER.
047905,000093:                                                                                                  #  6. ALL LISTS ARE COMBINATIONS OF CODED ERASABLE ADDRESS CONSTANTS
047906,000094:                                                                                                  #     CREATED FOR THE DOWNLIST PROGRAM.
047907,000095:                                                                                                  #     A. 1DNADR                  1-WORD DOWNLIST ADDRESS.
047908,000096:                                                                                                  #         SAME AS ECADR, BUT USED WHEN THE WORD ADDRESSED IS THE LEFT
047909,000097:                                                                                                  #         HALF OF A DOUBLE-PRECISION WORD FOR DOWN TELEMETRY.
047910,000098:                                                                                                  #     B. 2DNADR - 6DNADR         N-WORD DOWNLIST ADDRESS, N = 2 - 6.
047911,000099:                                                                                                  #         SAME AS 1DNADR, BUT WITH THE 4 UNUSED BITS OF THE ECADR FORMAT
047912,000100:                                                                                                  #         FILLED IN WITH 0001-0101.  USED TO POINT TO A LIST OF N DOUBLE-
047913,000101:                                                                                                  #         PRECISION WORDS, STORED CONSECUTIVELY, FOR DOWN TELEMETRY.
047914,000102:                                                                                                  #     C. DNCHAN                  DOWNLIST CHANNEL ADDRESS.
047915,000103:                                                                                                  #         SAME AS 1DNADR, BUT WITH PREFIX BITS 0111.  USED TO POINT TO
047916,000104:                                                                                                  #         A PAIR OF CHANNELS FOR DOWN TELEMETRY.
047917,000105:                                                                                                  #     D. DNPTR                   DOWN TELEMETRY SUBLIST POINTER.
047918,000106:                                                                                                  #         SAME AS CAF BUT TAGGED AS A CONSTANT. USED IN CONTROL LIST TO POINT TO A SUBLIST.
047919,000107:                                                                                                  #         CAUTION--- A DNPTR CANNOT BE USED IN A SUBLIST.
047920,000108:                                                                                                  #  7. THE WORD ORDER CODE IS SET TO ZERO AT THE BEGINNING OF EACH DOWNLIST (I.E. CONTROL LIST) AND WHEN
047921,000109:                                                                                                  #     A '1DNADR TIME2' IS DETECTED IN THE CONTROL LIST (ONLY).
047922,000110:                                                                                                  #  8. IN THE SNAPSHOT SUBLIST ONLY, THE DNADR'S CANNOT POINT TO THE FIRST WORD OF ANY EBANK.

048063,000251:                                                                                                  #  THE FOLLOWING CODING (FROM SNAPLOOP TO SNAPEND) IS FOR THE PURPOSE OF TAKING A SNAPSHOT OF 12 DP REGISTERS.
048064,000252:                                                                                                  #  THIS IS DONE BY SAVING 11 DP REGISTERS IN DNTMBUFF AND SENDING THE FIRST DP WORD IMMEDIATELY.
048065,000253:                                                                                                  #  THE SNAPSHOT PROCESSING IS THE MOST TIME CONSUMING AND THEREFORE THE CODING AND LIST STRUCTURE WERE DESIGNED
048066,000254:                                                                                                  #  TO MINIMIZE TIME. THE TIME OPTIMIZATION RESULTS IN RULES UNIQUE TO THE SNAPSHOT PORTION OF THE DOWNLIST.
048067,000255:                                                                                                  #  THESE RULES ARE......
048068,000256:                                                                                                  #        1.      ONLY 1DNADR'S CAN APPEAR IN THE SNAPSHOT SUBLIST
048069,000257:                                                                                                  #        2.      THE 1DNADR'S CANNOT REFER TO THE FIRST LOCATION IN ANY BANK.

048141,000329:                                                                                                  #  FUNCTIONAL DESCRIPTION - TO SEND (DUMP) ALL 8 BANKS OF ERASABLE STORAGE TWICE.  BANKS ARE SENT ONE AT A TIME
048142,000330:                                                                                                  #        EACH BANK IS PRECEEDED BY AN ID WORD, SYNCH BITS, ECADR AND TIME1 FOLLOWED BY THE 256D WORDS OF EACH
