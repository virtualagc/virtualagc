Source Code

These source-code files are part of a reconstructed copy of Luminary 178, the final release of the Apollo 14 Lunar Module (LM) Apollo Guidance Computer (AGC) software.

The reconstruction began with source code of Zerlina 56, Luminary 210, and Luminary 131, previously transcribed from digitized copies of those programs. The code was created by combining the three, using Zerlina 56 as a base, since it was based off of Luminary 183. The differences between the three were carefully merged using the Luminary memos as a guide, with all changes made after Luminary 178 being backed out. The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021152N. Note that page numbers in the reconstructed code match those for the baseline log section mentioned in each file's changelog; the page numbers for a real Luminary 178 listing would be different.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

004130,000026:                                                                                                  #        EQUALS  IS USED IN TWO WAYS.  IT IS OFTEN USED TO CHAIN A GROUP

004137,000033: 
004138,000034:                                                                                                  #                (X, Y, AND Z ARE CONSECUTIVE AND BEGIN AT START.        )
004139,000035:                                                                                                  #                (SIZE.X AND SIZE.Y ARE THE RESPECTIVE SIZES OF X AND Y,
004140,000036:                                                                                                  #                USUALLY NUMERIC, IE. 1, 2, 6, 18D ETC.                  )
004141,000037: 
004142,000038:                                                                                                  #        EQUALS  OFTEN IMPLIES THE SHARING OF REGISTERS (DIFFERENT NAMES

004145,000041:                                                                                                  #                        X       EQUALS  Y

004150,000046:                                                                                                  #                        X       =       Y

004152,000048:                                                                                                  #        THE SIZE AND UTILIZATION OF AN ERASABLE ARE OFTEN INCLUDED IN
004153,000049:                                                                                                  #        THE COMMENTS IN THE FOLLOWING FORM.  M(SIZE)N.
004154,000050: 
004155,000051:                                                                                                  #                M       REFERS TO THE MOBILITY OF THE ASSIGNMENT.
004156,000052:                                                                                                  #                        B       MEANS THAT THE SYMBOL IS REFERENCED BY BASIC
004157,000053:                                                                                                  #                                INSTRUCTIONS AND THUS IS E-BANK SENSITIVE.
004158,000054:                                                                                                  #                        I       MEANS THAT THE SYMBOL IS REFERENCED ONLY BY
004159,000055:                                                                                                  #                                INTERPRETIVE INSTRUCTIONS, AND IS THUS E-BANK
004160,000056:                                                                                                  #                                INSENSITIVE AND MAY APPEAR IN ANY E-BANK.
004161,000057: 
004162,000058:                                                                                                  #                SIZE    IS THE NUMBER OF REGISTERS INCLUDED BY THE SYMBOL.
004163,000059: 
004164,000060:                                                                                                  #                N       INDICATES THE NATURE OR PERMANENCE OF THE CONTENTS.
004165,000061:                                                                                                  #                        PL      MEANS THAT THE CONTENTS ARE PAD LOADED.
004166,000062:                                                                                                  #                        DSP     MEANS THAT THE REGISTER IS USED FOR A DISPLAY.
004167,000063:                                                                                                  #                        PRM     MEANS THAT THE REGISTER IS PERMANENT, IE. IT
004168,000064:                                                                                                  #                                IS USED DURING THE ENTIRE MISSION FOR ONE
004169,000065:                                                                                                  #                                PURPOSE AND CANNOT BE SHARED.
004170,000066:                                                                                                  #                        TMP     MEANS THAT THE REGISTER IS USED TEMPORARILY OR
004171,000067:                                                                                                  #                                IS A SCRATCH REGISTER FOR THE ROUTINE TO WHICH
004172,000068:                                                                                                  #                                IT IS ASSIGNED.  THAT IS, IT NEED NOT BE SET
004173,000069:                                                                                                  #                                PRIOR TO INVOCATION OF THE ROUTINE NOR DOES IT
004174,000070:                                                                                                  #                                CONTAIN USEFUL OUTPUT TO ANOTHER ROUTINE.  THUS
