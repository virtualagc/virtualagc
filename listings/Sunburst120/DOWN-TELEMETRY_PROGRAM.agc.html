Source Code

These source-code files were transcribed from scans made from Don Eyles's personal copy of BURST120 (SUNBURST 120). They were scanned at archive.org's Boston facility, and the scanning was sponsored by Mike Stewart. The code was transcribed from these scans by a team of volunteers who are referenced in the program comments. Comments from the original source code are in ALL-CAPS, whereas comments added later in transcription are in Mixed-Case. In some cases, where similar code blocks exist in previously-transcribed AGC programs (primarily Luminary 99, from Apollo 11) those code blocks were used as a starting point and then corrected to agree with the BURST120 scans. The full scans are available at the Virtual AGC project's collection at archive.org, while more-convenient reduced-size (but reduced-quality) images are available at the main Virtual AGC website. Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository. Notations on the program listing read, in part:

	YUL SYSTEM FOR AGC: REVISION 0 OF PROGRAM BURST120 
	BY NASA 2021106-031 DEC 7, 1967

	THIS LISTING IS A COPY OF A VERSION OF THE PROGRAM INTENDED 
	FOR USE IN THE ON-BOARD PRIMARY GUIDANCE COMPUTER IN THE 
	UNMANNED FLIGHT OF APOLLO LUNAR MODULE 1 --- THE AS206 MISSION.
Note that the date is the date of the printout, not the date of the program revision.

013579,000024:                                                                                                  #  ASSEMBLY- REV 12 OF SUNBURST
013580,000025:                                                                                                  #  FUNCTIONAL DESCRIPTION- THIS ROUTINE IS INITIATED BY TELEMETRY END
013581,000026:                                                                                                  #           PULSE FROM THE DOWNLINK TELEMETRY CONVERTER. THIS PULSE OCCURS
013582,000027:                                                                                                  #           AT 50 TIMES PER SEC(EVERY 20 MS) THEREFORE DODOWNTM IS
013583,000028:                                                                                                  #           EXECUTED AT THESE RATES. THIS ROUTINE SELECTS THE APPROPRIATE
013584,000029:                                                                                                  #           LGC DATA TO BE TRANSMITTED DOWNLINK AND LOADS IT INTO OUTPUT
013585,000030:                                                                                                  #           CHANNELS 34 AND 35. THE INFORMATION IS THEN GATED OUT FROM THE
013586,000031:                                                                                                  #           LGC IN SERIAL FASHION.
013587,000032:                                                                                                  #           THIS PROGRAM IS CODED FOR A 2 SECOND DOWNLIST. SINCE DOWNRUPTS
013588,000033:                                                                                                  #           OCCUR EVERY 20 MS AND 2 LGC COMPUTER WORDS CAN BE PLACED IN
013589,000034:                                                                                                  #           CHANNELS 34 AND 35 DURING EACH DOWNRUPT THE PROGRAM IS CAPABLE
013590,000035:                                                                                                  #           OF SENDING 200 LGC WORDS EVERY 2 SECONDS.
013591,000036:                                                                                                  #  CALLING SEQUENCE- NONE
013592,000037:                                                                                                  #           PROGRAM IS ENTERED VIA TCF DODOWNTM WHICH IS EXECUTED AS A
013593,000038:                                                                                                  #           RESULT OF A DOWNRUPT. CONTROL IS RETURNED VIA TCF RESUME WHICH
013594,000039:                                                                                                  #           IN EFFECT IS A RESUME.
013595,000040:                                                                                                  #  SUBROUTINES CALLED- NONE
013596,000041:                                                                                                  #  NORMAL EXIT MODE- TCF RESUME
013597,000042:                                                                                                  #  ALARM OR ABORT EXIT MODE- NONE
013598,000043:                                                                                                  #  RESTART PROTECTION- NONE.
013599,000044:                                                                                                  #           EXCEPT THAT WHEN A RESTART DOES OCCUR :STARTSUB: WILL
013600,000045:                                                                                                  #           INITIALIZE THE DOWNLIST POINTER TO THE BEGINNING OF THE NOMINAL
013601,000046:                                                                                                  #           DOWNLIST. THIS HAS THE EFFECT OF IGNORING THE REMAINDER OF THE
013602,000047:                                                                                                  #           DOWNLIST WHICH THE DOWN-TELEMETRY PROGRAM WAS WORKING ON WHEN
013603,000048:                                                                                                  #           THE RESTART OCCURRED.
013604,000049:                                                                                                  #  OUTPUT-  EVERY 2 SECONDS 100 DOUBLE PRECISION WORDS(I.E. 200 LGC
013605,000050:                                                                                                  #           COMPUTER WORDS) ARE TRANSMITTED VIA DOWNLINK.
013606,000051:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED- NONE
013607,000052:                                                                                                  #           DNTMGOTO,DNLSTADR AND EBANK ARE INITIALIZED IN STARTSUB(FRESH
013608,000053:                                                                                                  #           START AND RESTART)
013609,000054:                                                                                                  #  DEBRIS (ERASABLE LOCATIONS DESTROYED BY THIS PROGRAM)-
013610,000055:                                                                                                  #           LDATALST,ITEMP1,ITEMP2,DNTMBUFF TO DNTMBUFF + 21D,TMINDEX,DNQ
