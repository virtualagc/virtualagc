Note that the page images presented online are of reduced quality, and that higher-quality images are available. 055018,000032:                                                                                                  #  PROGRAM DESCRIPTION                           DATE   20 DECEMBER 1967
055019,000033:                                                                                                  #  PROGRAM NAME - SELF-CHECK                     LOG SECTION  AGC BLOCK TWO SELF-CHECK
055020,000034:                                                                                                  #  MOD NO - 1                                    ASSEMBLY SUBROUTINE UTILITYM REV 25
055021,000035:                                                                                                  #  MOD BY - GAUNTT
055022,000036: 
055023,000037:                                                                                                  #  FUNCTIONAL DESCRIPTION
055024,000038:                                                                                                  #        PROGRAM HAS TWO MAIN PARTS.  THE FIRST IS SELF-CHECK WHICH RUNS AS A ZERO PRIORITY JOB WITH NO CORE SET, AS
055025,000039:                                                                                                  #        PART OF THE BACK-UP IDLE LOOP.  THE SECOND IS SHOW-BANKSUM WHICH RUNS AS A REGULAR EXECUTIVE JOB WITH ITS OWN
055026,000040:                                                                                                  #        STARTING VERB.
055027,000041: 
055028,000042:                                                                                                  #        THE PURPOSE OF SELF-CHECK IS TO CHECK OUT VARIOUS PARTS OF THE COMPUTER AS OUTLINED BELOW IN THE OPTIONS.
055029,000043: 
055030,000044:                                                                                                  #        THE PURPOSE OF SHOW-BANKSUM IS TO DISPLAY THE SUM OF EACH BANK, ONE AT A TIME.
055031,000045: 
055032,000046:                                                                                                  #        IN ALL THERE ARE 7 POSSIBLE OPTIONS IN THIS BLOCK II VERSION OF SELF-CHECK.  MORE DETAIL DESCRIPTION MAY BE
055033,000047:                                                                                                  #        FOUND IN E-2065 BLOCK II AGC SELF-CHECK AND SHOW BANKSUM BY EDWIN D. SMALLY DECEMBER 1966, AND ADDENDA 2 AND 3.
055034,000048: 
055035,000049:                                                                                                  #        THE DIFFERENT OPTIONS ARE CONTROLLED BY PUTTING DIFFERENT NUMBERS IN THE SMODE REGISTER (NOUN 27).  BELOW IS
055036,000050:                                                                                                  #        A DESCRIPTION OF WHAT PARTS OF THE COMPUTER THAT ARE CHECKED BY THE OPTIONS, AND THE CORRESPONDING NUMBER, IN
055037,000051:                                                                                                  #        OCTAL, TO LOAD INTO SMODE.
055038,000052:                                                                                                  #                +-4             ERASABLE MEMORY
055039,000053:                                                                                                  #                +-5             FIXED MEMORY
055040,000054:                                                                                                  #                +-1,2,3,6,7,10  EVERYTHING IN OPTIONS 4 AND 5.
055041,000055:                                                                                                  #                -0              SAME AS +-10 UNTIL AN ERROR IS DETECTED.
055042,000056:                                                                                                  #                +0              NO CHECK, PUTS COMPUTER INTO THE BACKUP IDLE LOOP.
055043,000057: 
055044,000058:                                                                                                  #  WARNINGS
055045,000059:                                                                                                  #        USE OF E MEMORY RESERVED FOR SELF-CHECK (EVEN IN IDLE LOOP) AS TEMP STORAGE BY OTHER PROGRAMS IS DANGEROUS.
055046,000060:                                                                                                  #        SMODE SET GREATER THAN OCT 10 PUTS COMPUTER INTO BACKUP IDLE LOOP.
055047,000061: 
055048,000062:                                                                                                  #  CALLING SEQUENCE
055049,000063:                                                                                                  #        TO CALL SELF-CHECK KEY IN
055050,000064:                                                                                                  #                V 21 N 27 E     OPTION NUMBER E
055051,000065:                                                                                                  #        TO CALL SHOW-BANKSUM KEY IN
055052,000066:                                                                                                  #                V 91 E          DISPLAYS FIRST BANK
055053,000067:                                                                                                  #                V 33 E          PROCEED, DISPLAYS NEXT BANK
055054,000068: 
055055,000069:                                                                                                  #  EXIT MODES, NORMAL AND ALARM
055056,000070:                                                                                                  #        SELF-CHECK NORMALLY CONTINUES INDEFINITELY UNLESS THERE IS AN ERROR DETECTED.  IF SO + OPTION NUMBERS PUT
055057,000071:                                                                                                  #        COMPUTER INTO BACKUP IDLE LOOP, - OPTION NUMBERS RESTART THE OPTION.
055058,000072: 
055059,000073:                                                                                                  #        THE -0 OPTION PROCEEDS FROM THE LINE FOLLOWING THE LINE WHERE THE ERROR WAS DETECTED.
055060,000074: 
055061,000075:                                                                                                  #        SHOW-BANKSUM PROCEEDS UNTIL A TERMINATE IS KEYED IN (V 34 E).  THE COMPUTER IS PUT INTO THE BACKUP IDLE LOOP
055062,000076: 
055063,000077:                                                                                                  #  OUTPUT

Page 1285 

055065,000079:                                                                                                  #        SELF-CHECK UPON DETECTING AN ERROR LOADS THE SELF-CHECK ALARM CONSTANT (01102) INTO THE FAILREG SET AND
055066,000080:                                                                                                  #        TURNS ON THE ALARM LIGHT.  THE OPERATOR MAY THEN DISPLAY THE THREE FAILREGS BY KEYING IN V 05 N 09 E.  FOR FURTHER
055067,000081:                                                                                                  #        INFORMATION HE MAY KEY IN V 05 N 08 E, THE DSKY DISPLAY IN R1 WILL BE ADDRESS+1 OF WHERE THE ERROR WAS DETECTED,
055068,000082:                                                                                                  #        IN R2 THE BBCON OF SELF-CHECK, AND IN R3 THE TOTAL NUMBER OF ERRORS DETECTED BY SELF-CHECK SINCE THE LAST MAN
055069,000083:                                                                                                  #        INITIATED FRESH START (SLAP1).
055070,000084: 
055071,000085:                                                                                                  #        SHOW-BANKSUM STARTING WITH BANK 0 DISPLAYS IN R1 THE BANK SUM (A +-NUMBER EQUAL TO THE BANK NUMBER), IN R2
055072,000086:                                                                                                  #        THE BANK NUMBER, AND IN R3 THE BUGGER WORD.
055073,000087: 
055074,000088:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
055075,000089:                                                                                                  #        ACCOMPLISHED BY FRESH START
055076,000090:                                                                                                  #                SMODE SET TO +0
055077,000091: 
055078,000092:                                                                                                  #  DEBRIS
055079,000093:                                                                                                  #        ALL EXITS FROM THE CHECK OF ERASABLE (ERASCHK) RESTORE ORIGINAL CONTENTS TO REGISTERS UNDER CHECK.
055080,000094:                                                                                                  #        EXCEPTION IS A RESTART.  RESTART THAT OCCURS DURING ERASCHK RESTORES ERASABLE, UNLESS THERE IS EVIDENCE TO DOUBT
055081,000095:                                                                                                  #        E MEMORY, IN WHICH CASE PROGRAM THEN DOES A FRESH START (DOFSTART).
