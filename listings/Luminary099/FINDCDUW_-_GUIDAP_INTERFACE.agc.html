Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

039794,000034:                                                                                                  #  PROGRAM NAME:  FINDCDUW
039795,000035:                                                                                                  #  MOD NUMBER:  1        68 07 15
039796,000036:                                                                                                  #  MOD AUTHOR:  KLUMPP
039797,000037: 
039798,000038:                                                                                                  #  OBJECTS OF MOD:       1.      TO SUPPLY COMMANDED GIMBAL ANGLES FOR NOUN 22.
039799,000039:                                                                                                  #                        2.      TO MAINTAIN CORRECT AND CURRENT THRUST
039800,000040:                                                                                                  #                                DIRECTION DATA IN ALL MODES.  THIS IS DONE BY
039801,000041:                                                                                                  #                                FETCHING FOR THE THRUST DIRECTION FILTER THE
039802,000042:                                                                                                  #                                CDUD'S IN PNGCS-AUTO, THE CDU'S IN ALL OTHER
039803,000043:                                                                                                  #                                MODES.
039804,000044:                                                                                                  #                        3.      TO SUBSTITUTE A STOPRATE FOR THE NORMAL
039805,000045:                                                                                                  #                                AUTOPILOT COMMANDS WHENEVER
039806,000046:                                                                                                  #                                1) NOT IN PNGCS-AUTO, OR
039807,000047:                                                                                                  #                                2) ENGINE IS OFF.
039808,000048: 
039809,000049:                                                                                                  #  FUNCTIONAL DESCRIPTION:
039810,000050: 
039811,000051:                                                                                                  #  FINDCDUW PROVIDES THE INTERFACES BETWEEN THE VARIOUS POWERED FLITE GUIDANCE PROGRAMS
039812,000052:                                                                                                  #  AND THE DIGITAL AUTOPILOT.  THE INPUTS TO FINDCDUW ARE THE THRUST COMMAND VECTOR
039813,000053:                                                                                                  #  AND THE WINDOW COMMAND VECTOR, AND THE OUTPUTS ARE THE GIMBAL ANGLE
039814,000054:                                                                                                  #  INCREMENTS, THE COMMANDED ATTITUDE ANGLE RATES, AND THE COMMANDED
039815,000055:                                                                                                  #  ATTITUDE LAG ANGLES (WHICH ACCOUNT FOR THE ANGLES BY WHICH THE BODY WILL
039816,000056:                                                                                                  #  LAG BEHIND A RAMP COMMAND IN ATTITUDE ANGLE DUE TO THE FINITE ANGULAR
039817,000057:                                                                                                  #  ACCELERATIONS AVAILABLE).
039818,000058: 
039819,000059:                                                                                                  #  FINDCDUW ALINES THE ESTIMATED THRUST VECTOR FROM THE THRUST DIRECTION
039820,000060:                                                                                                  #  FILTER WITH THE THRUST COMMAND VECTOR, AND, WHEN XOVINHIB SET,
039821,000061:                                                                                                  #  ALINES THE +Z HALF OF THE LM ZX PLANE WITH THE WINDOW COMMAND VECTOR.

039824,000064:                                                                                                  #  SPECIFICATIONS:
039825,000065: 
039826,000066:                                                                                                  #  INITIALIZATION:       A SINGLE INTERPRETIVE CALL TO INITCDUW IS REQUIRED
039827,000067:                                                                                                  #                        BEFORE EACH GUIDED MANEUVER USING FINDCDUW.
039828,000068: 
039829,000069:                                                                                                  #  CALL:                 INTERPRETIVE CALL TO FINDCDUW WITH THE THRUST COMMAND
039830,000070:                                                                                                  #                        VECTOR IN MPAC.  INTERPRETIVE CALL TO FINDCDUW -2 WITH
039831,000071:                                                                                                  #                        THE THRUST COMMAND VECTOR IN UNFC/2 AND NOT IN MPAC.
039832,000072: 
039833,000073:                                                                                                  #  RETURNS:              NORMAL INTERPRETIVE IN ALL CASES
039834,000074:                                                                                                  #                        1.      NORMALLY ALL AUTOPILOT CMDS ARE ISSUED.
039835,000075:                                                                                                  #                        2.      IF NOT PNGCS AUTO, DO STOPRATE AND RETURN
039836,000076:                                                                                                  #                                WITHOUT ISSUING AUTOPILOT CMDS.
039837,000077:                                                                                                  #                        3.      IF ENGINE OFF, DO STOPRATE AND RETURN WITHOUT
039838,000078:                                                                                                  #                                ISSUING AUTOPILOT CMDS.
039839,000079: 
039840,000080:                                                                                                  #  ALARMS:               00401   IF INPUTS DETERMINE AN ATTITUDE IN GIMBAL LOCK.
039841,000081:                                                                                                  #                                FINDCDUW DRIVES CDUXD AND CDUYD TO THE RQD VALUES,
039842,000082:                                                                                                  #                                BUT DRIVES CDUZD ONLY TO THE GIMBAL LOCK CONE.
039843,000083: 
039844,000084:                                                                                                  #                        00402   IF UNFC/2 OR UNWC/2 PRODUCE OVERFLOW WHEN
039845,000085:                                                                                                  #                                UNITIZED USING NORMUNIT.  FINDCDUW ISSUES
039846,000086:                                                                                                  #                                STOPRATE AS ONLY INPUT TO AUTOPILOT.
039847,000087: 
039848,000088:                                                                                                  #  INPUTS:               UNFC/2          THRUST COMMAND VECTOR, NEED NOT BE SEMI-UNIT.
039849,000089:                                                                                                  #                        UNWC/2          WINDOW COMMAND VECTOR, NEED NOT BE SEMI-UNIT.
039850,000090:                                                                                                  #                        XOVINHIB        FLAG DENOTING X AXIS OVERRIDE INHIBITED.
039851,000091:                                                                                                  #                        CSMDOCKD        FLAG DENOTING CSM DOCKED.
039852,000092:                                                                                                  #                        STEERSW         FLAG DENOTING INSUFF THRUST FOR THRUST DIR FLTR.
039853,000093: 
039854,000094:                                                                                                  #  OUTPUTS:              DELCDUX,Y,Z
039855,000095:                                                                                                  #                        OMEGAPD,+1,+2
039856,000096:                                                                                                  #                        DELPEROR,+1,+2
039857,000097:                                                                                                  #                        CPHI,+1,+2 FOR NOUN22
039858,000098: 
039859,000099:                                                                                                  #  DEBRIS:               FINDCDUW DESTROYS SINCDUX,Y,Z AND COSCDUX,Y,Z BY
039860,000100:                                                                                                  #                        WRITING INTO THESE LOCATIONS THE SINES AND COSINES
039861,000101:                                                                                                  #                        OF THE CDUD'S IN PNGCS-AUTO, OF THE CDU'S OTHERWISE.
