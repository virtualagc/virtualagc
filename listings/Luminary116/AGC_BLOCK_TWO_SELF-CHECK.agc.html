Source Code

These source-code files derive from a printout of Luminary 116 (the Apollo 12 Lunar Module guidance computer program), from the personal library of original AGC developer Don Eyles, digitally photographed at archive.org, financially sponsored by Ron Burkey, and transcribed to source code by a team of volunteers. This colorized, syntax-highlighted form was created by assembling that transcribed source code. Note that the full page images are available on the Virtual AGC project page at archive.org, while reduced-size images are presented at the VirtualAGC project website. Report or fix any transcription errors at the Virtual AGC project code repository.

Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 116 OF AGC PROGRAM LUMINARY BY NASA 2021112-071
	19:09 AUG. 11,1969
Note that the date is the date of the printout, not the date of the program revision.

054015,000023:                                                                                                  #  PROGRAM NAME - SELF-CHECK                     LOG SECTION  AGC BLOCK TWO SELF-CHECK

054019,000027:                                                                                                  #  FUNCTIONAL DESCRIPTION
054020,000028:                                                                                                  #       PROGRAM HAS TWO MAIN PARTS. THE FIRST IS SELF-CHECK WHICH RUNS AS A ZERO PRIORITY JOB WITH NO CORE SET, AS
054021,000029:                                                                                                  #  PART OF THE BACK-UP IDLE LOOP. THE SECOND IS SHOW-BANKSUM WHICH RUNS AS A REGULAR EXECUTIVE JOB WITH ITS OWN
054022,000030:                                                                                                  #  STARTING VERB.
054023,000031:                                                                                                  #       THE PURPOSE OF SELF-CHECK IS TO CHECK OUT VARIOUS PARTS OF THE COMPUTER AS OUTLINED BELOW IN THE OPTIONS.
054024,000032:                                                                                                  #       THE PURPOSE OF SHOW-BANKSUM IS TO DISPLAY THE SUM OF EACH BANK, ONE AT A TIME.
054025,000033:                                                                                                  #       IN ALL THERE ARE 7 POSSIBLE OPTIONS IN THIS BLOCK II VERSION OF SELF-CHECK. MORE DETAIL DESCRIPTION MAY BE
054026,000034:                                                                                                  #  FOUND IN E-2065 BLOCK II AGC SELF-CHECK AND SHOW BANKSUM BY EDWIN D. SMALLY DECEMBER 1966, AND ADDENDA 2 AND 3.
054027,000035:                                                                                                  #       THE DIFFERENT OPTIONS ARE CONTROLLED BY PUTTING DIFFERENT NUMBERS IN THE SMODE REGISTER (NOUN 27). BELOW IS
054028,000036:                                                                                                  #  A DESCRIPTION OF WHAT PARTS OF THE COMPUTER THAT ARE CHECKED BY THE OPTIONS, AND THE CORRESPONDING NUMBER, IN
054029,000037:                                                                                                  #  OCTAL, TO LOAD INTO SMODE.
054030,000038:                                                                                                  #  +-4   ERASABLE MEMORY
054031,000039:                                                                                                  #  +-5   FIXED MEMORY
054032,000040:                                                                                                  #  +-1,2,3,6,7,10   EVERYTHING IN OPTIONS 4 AND 5.
054033,000041:                                                                                                  #  -0    SAME AS +-10 UNTIL AN ERROR IS DETECTED.
054034,000042:                                                                                                  #  +0    NO CHECK, PUTS COMPUTER INTO THE BACKUP IDLE LOOP.
054035,000043: 
054036,000044:                                                                                                  #  WARNINGS
054037,000045: 
054038,000046:                                                                                                  #       USE OF E MEMORY RESERVED FOR SELF-CHECK (EVEN IN IDLE LOOP) AS TEMP STORAGE BY OTHER PROGRAMS IS DANGEROUS.
054039,000047:                                                                                                  #       SMODE SET GREATER THAN OCT 10 PUTS COMPUTER INTO BACKUP IDLE LOOP.
054040,000048: 
054041,000049:                                                                                                  #  CALLING SEQUENCE
054042,000050: 
054043,000051:                                                                                                  #       TO CALL SELF-CHECK KEY IN
054044,000052:                                                                                                  #            V 21 N 27 E  OPTION NUMBER E
054045,000053:                                                                                                  #        TO CALL SHOW-BANKSUM KEY IN
054046,000054:                                                                                                  #            V 91 E          DISPLAYS FIRST BANK
054047,000055:                                                                                                  #            V 33 E          PROCEED, DISPLAYS NEXT BANK
054048,000056: 
054049,000057:                                                                                                  #  EXIT MODES, NORMAL AND ALARM
054050,000058: 
054051,000059:                                                                                                  #       SELF-CHECK NORMALLY CONTINUES INDEFINITELY UNLESS THERE IS AN ERROR DETECTED. IF SO + OPTION NUMBERS PUT
054052,000060:                                                                                                  #  COMPUTER INTO BACKUP IDLE LOOP, - OPTION NUMBERS RESTART THE OPTION.
054053,000061: 
054054,000062:                                                                                                  #       THE -0 OPTION PROCEEDS FROM THE LINE FOLLOWING THE LINE WHERE THE ERROR WAS DETECTED.
054055,000063: 
054056,000064:                                                                                                  #       SHOW-BANKSUM PROCEEDS UNTIL A TERMINATE IS KEYED IN (V 34 E).  THE COMPUTER IS PUT INTO THE BACKUP IDLE LOOP
054057,000065: 
054058,000066:                                                                                                  #  OUTPUT

054060,000068:                                                                                                  #       SELF-CHECK UPON DETECTING AN ERROR LOADS THE SELF-CHECK ALARM CONSTANT (01102) INTO THE FAILREG SET AND
054061,000069:                                                                                                  #  TURNS ON THE ALARM LIGHT. THE OPERATOR MAY THEN DISPLAY THE THREE FAILREGS BY KEYING IN V 05 N 09 E. FOR FURTHER
054062,000070:                                                                                                  #  INFORMATION HE MAY KEY IN V 05 N 08 E, THE DSKY DISPLAY IN R1 WILL BE ADDRESS+1 OF WHERE THE ERROR WAS DETECTED,
054063,000071:                                                                                                  #  IN R2 THE BBCON OF SELF-CHECK, AND IN R3 THE TOTAL NUMBER OF ERRORS DETECTED BY SELF-CHECK SINCE THE LAST MAN
054064,000072:                                                                                                  #  INITIATED FRESH START (SLAP1).
054065,000073:                                                                                                  #       SHOW-BANKSUM STARTING WITH BANK 0 DISPLAYS IN R1 THE BANK SUM (A +-NUMBER EQUAL TO THE BANK NUMBER), IN R2
054066,000074:                                                                                                  #  THE BANK NUMBER, AND IN R3 THE BUGGER WORD.
054067,000075: 
054068,000076:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
054069,000077: 
054070,000078:                                                                                                  #       ACCOMPLISHED BY FRESH START
054071,000079:                                                                                                  #            SMODE SET TO +0
054072,000080: 
054073,000081:                                                                                                  #  DEBRIS
054074,000082: 
054075,000083:                                                                                                  #        ALL EXITS FROM THE CHECK OF ERASABLE (ERASCHK) RESTORE ORIGINAL CONTENTS TO REGISTERS UNDER CHECK.
054076,000084:                                                                                                  #  EXCEPTION IS A RESTART. RESTART THAT OCCURS DURING ERASCHK RESTORES ERASABLE, UNLESS THERE IS EVIDENCE TO DOUBT
054077,000085:                                                                                                  #  E MEMORY, IN WHICH CASE PROGRAM THEN DOES A FRESH START (DOFSTART).
