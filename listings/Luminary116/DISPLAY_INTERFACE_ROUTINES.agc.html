Source Code

These source-code files derive from a printout of Luminary 116 (the Apollo 12 Lunar Module guidance computer program), from the personal library of original AGC developer Don Eyles, digitally photographed at archive.org, financially sponsored by Ron Burkey, and transcribed to source code by a team of volunteers. This colorized, syntax-highlighted form was created by assembling that transcribed source code. Note that the full page images are available on the Virtual AGC project page at archive.org, while reduced-size images are presented at the VirtualAGC project website. Report or fix any transcription errors at the Virtual AGC project code repository.

Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 116 OF AGC PROGRAM LUMINARY BY NASA 2021112-071
	19:09 AUG. 11,1969
Note that the date is the date of the printout, not the date of the program revision.

056439,000024:                                                                                                  #  DISPLAYS CAN BE CLASSIFIED INTO THE FOLLOWING CATEGORIES-
056440,000025: 
056441,000026:                                                                                                  #      1.  PRIORITY DISPLAYS- DISPLAYS WHICH TAKE PRIORITY OVER ALL OTHER DISPLAYS. USUALLY THESE DISPLAYS ARE SENT
056442,000027:                                                                                                  #          OUT UNDER CRITICAL ALARM CONDITIONS.
056443,000028:                                                                                                  #      2.  EXTENDED VERB DISPLAYS- ALL EXTENDED VERBS AND MARK ROUTINES SHOULD USE EXTENDED VERB (MARK) DISPLAYS.
056444,000029:                                                                                                  #      3.  NORMAL DISPLAYS- ALL MISSION PROGRAM DISPLAYS WHICH INTERFACE WITH THE ASTRONAUT DURING THE NORMAL
056445,000030:                                                                                                  #          SEQUENCE OF EVENTS.
056446,000031:                                                                                                  #      4.  MISC. DISPLAYS- ALL DISPLAYS NOT HANDLED BY THE DISPLAY INTERFACE ROUTINES. THESE INCLUDE SUCH DISPLAYS AS
056447,000032:                                                                                                  #          MM DISPLAYS AND SPECIAL PURPOSE DISPLAYS HANDLED BY PINBALL.
056448,000033: 
056449,000034:                                                                                                  #      5.  ASTRONAUT INITIATED DISPLAYS- ALL DISPLAYS INITIATED EXTERNALLY.
056450,000035: 
056451,000036:                                                                                                  #  THE FOLLOWING TERMS ARE USED TO DESCRIBE THE STATUS OF DISPLAYS-
056452,000037: 
056453,000038:                                                                                                  #      1.  ACTIVE-THE DISPLAY WHICH IS (1) BEING DISPLAYED TO THE ASTRONAUT AND WAITING FOR A RESPONSE OR
056454,000039:                                                                                                  #          (2) WAITING FIRST IN LINE FOR THE ASTRONAUT TO FINISH USING THE DSKY OR (3) BEING DISPLAYED ON THE DSKY
056455,000040:                                                                                                  #          BUT NOT WAITING FOR A RESPONSE.
056456,000041:                                                                                                  #      2.  INACTIVE-A DISPLAY WHICH HAS (1) BEEN ACTIVE BUT WAS INTERRUPTED BY A DISPLAY OF HIGHER PRIORITY,
056457,000042:                                                                                                  #          (2) BEEN PUT INTO THE WAITING LIST AT TIME IT WAS REQUESTED DUE TO THE FACT A HIGHER PRIORITY DISPLAY
056458,000043:                                                                                                  #          WAS ALREADY GOING, (3) BEEN INTERRUPTED BY THE ASTRONAUT (CALLED A PINBRANCH CONDITION, SINCE THIS TYPE
056459,000044:                                                                                                  #          OF INACTIVE DISPLAY IS USUALLY REACTIVATED ONLY BY PINBALL) OR (4) A DISPLAY WHICH HAS FINISHED BUT STILL
056460,000045:                                                                                                  #          HAS INFO SAVED FOR RESTART PURPOSES.
056461,000046: 
056462,000047:                                                                                                  #  DISPLAY PRIORITIES WORK AS FOLLOWS-
056463,000048: 
056464,000049:                                                                                                  #    INTERRUPTS-
056465,000050:                                                                                                  #      1.  THE ASTRONAUT CAN INTERRUPT ANY DISPLAY WITH AN EXTERNAL DISPLAY REQUEST.
056466,000051:                                                                                                  #      2.  INTERNAL DISPLAYS CAN NOT BE SENT OUT WHEN THE ASTRONAUT IS USING THE DSKY.
056467,000052:                                                                                                  #      3.  PRIORITY DISPLAYS INTERRUPT ALL OTHER TYPES OF INTERNAL DISPLAYS. A PRIORITY DISPLAY INTERRUPTING ANOTHER
056468,000053:                                                                                                  #          PRIORITY DISPLAY WILL CAUSE AN ABORT UNLESS BIT14 IS SET FOR THE LINUS ROUTINE.
056469,000054:                                                                                                  #      4.  A MARK DISPLAY INTERRUPTS ANY NORMAL DISPLAY.
056470,000055:                                                                                                  #      5.  A MARK THAT INTERRUPTS A MARK COMPLETELY REPLACES IT.
056471,000056: 
056472,000057:                                                                                                  #    ORDER OF WAITING DISPLAYS-
056473,000058:                                                                                                  #      1.  ASTRONAUT EXTERNAL USE
056474,000059:                                                                                                  #      2.  PRIORITY
056475,000060:                                                                                                  #      3.  INTERRUPTED MARK
056476,000061:                                                                                                  #      4.  INTERRUPTED NORMAL

056478,000063:                                                                                                  #      5.  MARK TO BE REQUESTED (SEE DESCRIPTION OF ENDMARK)
056479,000064:                                                                                                  #      6.  MARK WAITING
056480,000065:                                                                                                  #      7.  NORMAL WAITING

056483,000068:                                                                                                  #  THE DISPLAY ROUTINES ARE INTENDED TO SERVE AS AN INTERFACE BETWEEN THE USER AND PINBALL.  THE
056484,000069:                                                                                                  #  FOLLOWING STATEMENTS CAN BE MADE ABOUT NORMAL DISPLAYS AND PRIORITY DISPLAYS (A DESCRIPTION OF MARK ROUTINES
056485,000070:                                                                                                  #  WILL FOLLOW LATER):
056486,000071: 
056487,000072:                                                                                                  #       1.  ALL ROUTINES THAT END IN R HAVE AN IMMEDIATE RETURN TO THE USER.  FOR ALL FLASHING DISPLAYS THIS RETURN
056488,000073:                                                                                                  #           IS TO THE USERS CALL CADR +4.  FOR THE ONLY NON FLASHING IMMEDIATE RETURN DISPLAY (GODSPR) THIS RETURN
056489,000074:                                                                                                  #           IS TO THE USERS CALLING LOC +1.
056490,000075:                                                                                                  #       2.  ALL ROUTINES NOT ENDING IN R DO NOT DO AN IMMEDIATE RETURN TO THE USER.
056491,000076:                                                                                                  #       3.  ALL ROUTINES THAT END IN R START A SEPARATE JOB (MAKEPLAY) WITH USERS JOB PRIORITY.
056492,000077:                                                                                                  #       4.  ALL ROUTINES NOT ENDING IN R BRANCH DIRECTLY TO MAKEPLAY WHICH MAKES THESE DISPLAYS A PART OF THE
056493,000078:                                                                                                  #           USERS JOB.
056494,000079:                                                                                                  #       5.  ALL DISPLAY ROUTINES ARE CALLED VIA BANKCALL.
056495,000080:                                                                                                  #       6.  TO RESTART A DISPLAY THE USER WILL GENERALLY USE A PHASE OF ONE WITH DESIRED RESTART GROUP (SEE
056496,000081:                                                                                                  #           DESCRIPTION OF RESTARTS).
056497,000082:                                                                                                  #       7.  ALL FLASHING DISPLAYS HAVE 3 RETURNS TO THE USER FROM ASTRONAUT RESPONSES.  A TERMINATE (V34) BRANCHES
056498,000083:                                                                                                  #           TO THE USERS CALL CADR +1.  A PROCEED (V33) BRANCHES TO THE USERS CALL CADR +2.  AN ENTER OR RECYCLE
056499,000084:                                                                                                  #           (V32) BRANCHES TO THE USERS CALL CADR +3.
