Source Code

These source-code files derive from a printout of Luminary 116 (the Apollo 12 Lunar Module guidance computer program), from the personal library of original AGC developer Don Eyles, digitally photographed at archive.org, financially sponsored by Ron Burkey, and transcribed to source code by a team of volunteers. This colorized, syntax-highlighted form was created by assembling that transcribed source code. Note that the full page images are available on the Virtual AGC project page at archive.org, while reduced-size images are presented at the VirtualAGC project website. Report or fix any transcription errors at the Virtual AGC project code repository.

Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 116 OF AGC PROGRAM LUMINARY BY NASA 2021112-071
	19:09 AUG. 11,1969
Note that the date is the date of the printout, not the date of the program revision.

039048,000025:                                                                                                  #  PROGRAM NAME:  FINDCDUW
039049,000026:                                                                                                  #  MOD NUMBER:  1        68 07 15
039050,000027:                                                                                                  #  MOD AUTHOR:  KLUMPP
039051,000028: 
039052,000029:                                                                                                  #  OBJECTS OF MOD:       1.      TO SUPPLY COMMANDED GIMBAL ANGLES FOR NOUN 22.
039053,000030:                                                                                                  #                        2.      TO MAINTAIN CORRECT AND CURRENT THRUST
039054,000031:                                                                                                  #                                DIRECTION DATA IN ALL MODES.  THIS IS DONE BY
039055,000032:                                                                                                  #                                FETCHING FOR THE THRUST DIRECTION FILTER THE
039056,000033:                                                                                                  #                                CDUD'S IN PNGCS-AUTO, THE CDU'S IN ALL OTHER
039057,000034:                                                                                                  #                                MODES.
039058,000035:                                                                                                  #                        3.      TO SUBSTITUTE A STOPRATE FOR THE NORMAL
039059,000036:                                                                                                  #                                AUTOPILOT COMMANDS WHENEVER
039060,000037:                                                                                                  #                                1) NOT IN PNGCS-AUTO, OR
039061,000038:                                                                                                  #                                2) ENGINE IS OFF.
039062,000039: 
039063,000040:                                                                                                  #  FUNCTIONAL DESCRIPTION:
039064,000041: 
039065,000042:                                                                                                  #  FINDCDUW PROVIDES THE INTERFACES BETWEEN THE VARIOUS POWERED FLITE GUIDANCE PROGRAMS
039066,000043:                                                                                                  #  AND THE DIGITAL AUTOPILOT.  THE INPUTS TO FINDCDUW ARE THE THRUST COMMAND VECTOR
039067,000044:                                                                                                  #  AND THE WINDOW COMMAND VECTOR, AND THE OUTPUTS ARE THE GIMBAL ANGLE
039068,000045:                                                                                                  #  INCREMENTS, THE COMMANDED ATTITUDE ANGLE RATES, AND THE COMMANDED
039069,000046:                                                                                                  #  ATTITUDE LAG ANGLES (WHICH ACCOUNT FOR THE ANGLES BY WHICH THE BODY WILL
039070,000047:                                                                                                  #  LAG BEHIND A RAMP COMMAND IN ATTITUDE ANGLE DUE TO THE FINITE ANGULAR
039071,000048:                                                                                                  #  ACCELERATIONS AVAILABLE).
039072,000049: 
039073,000050:                                                                                                  #  FINDCDUW ALINES THE ESTIMATED THRUST VECTOR FROM THE THRUST DIRECTION
039074,000051:                                                                                                  #  FILTER WITH THE THRUST COMMAND VECTOR, AND, WHEN XOVINHIB SET,
039075,000052:                                                                                                  #  ALINES THE +Z HALF OF THE LM ZX PLANE WITH THE WINDOW COMMAND VECTOR.

039078,000055:                                                                                                  #  SPECIFICATIONS:
039079,000056: 
039080,000057:                                                                                                  #  INITIALIZATION:       A SINGLE INTERPRETIVE CALL TO INITCDUW IS REQUIRED
039081,000058:                                                                                                  #                        BEFORE EACH GUIDED MANEUVER USING FINDCDUW.
039082,000059: 
039083,000060:                                                                                                  #  CALL:                 INTERPRETIVE CALL TO FINDCDUW WITH THE THRUST COMMAND
039084,000061:                                                                                                  #                        VECTOR IN MPAC.  INTERPRETIVE CALL TO FINDCDUW -2 WITH
039085,000062:                                                                                                  #                        THE THRUST COMMAND VECTOR IN UNFC/2 AND NOT IN MPAC.
039086,000063: 
039087,000064:                                                                                                  #  RETURNS:              NORMAL INTERPRETIVE IN ALL CASES
039088,000065:                                                                                                  #                        1.      NORMALLY ALL AUTOPILOT CMDS ARE ISSUED.
039089,000066:                                                                                                  #                        2.      IF NOT PNGCS AUTO, DO STOPRATE AND RETURN
039090,000067:                                                                                                  #                                WITHOUT ISSUING AUTOPILOT CMDS.
039091,000068:                                                                                                  #                        3.      IF ENGINE OFF, DO STOPRATE AND RETURN WITHOUT
039092,000069:                                                                                                  #                                ISSUING AUTOPILOT CMDS.
039093,000070: 
039094,000071:                                                                                                  #  ALARMS:               00401   IF INPUTS DETERMINE AN ATTITUDE IN GIMBAL LOCK.
039095,000072:                                                                                                  #                                FINDCDUW DRIVES CDUXD AND CDUYD TO THE RQD VALUES,
039096,000073:                                                                                                  #                                BUT DRIVES CDUZD ONLY TO THE GIMBAL LOCK CONE.
039097,000074: 
039098,000075:                                                                                                  #                        00402   IF UNFC/2 OR UNWC/2 PRODUCE OVERFLOW WHEN
039099,000076:                                                                                                  #                                UNITIZED USING NORMUNIT.  FINDCDUW ISSUES
039100,000077:                                                                                                  #                                STOPRATE AS ONLY INPUT TO AUTOPILOT.
039101,000078: 
039102,000079:                                                                                                  #  INPUTS:               UNFC/2          THRUST COMMAND VECTOR, NEED NOT BE SEMI-UNIT.
039103,000080:                                                                                                  #                        UNWC/2          WINDOW COMMAND VECTOR, NEED NOT BE SEMI-UNIT.
039104,000081:                                                                                                  #                        XOVINHIB        FLAG DENOTING X AXIS OVERRIDE INHIBITED.
039105,000082:                                                                                                  #                        CSMDOCKD        FLAG DENOTING CSM DOCKED.
039106,000083:                                                                                                  #                        STEERSW         FLAG DENOTING INSUFF THRUST FOR THRUST DIR FLTR.
039107,000084: 
039108,000085:                                                                                                  #  OUTPUTS:              DELCDUX,Y,Z
039109,000086:                                                                                                  #                        OMEGAPD,+1,+2
039110,000087:                                                                                                  #                        DELPEROR,+1,+2
039111,000088:                                                                                                  #                        CPHI,+1,+2 FOR NOUN22
039112,000089: 
039113,000090:                                                                                                  #  DEBRIS:               FINDCDUW DESTROYS SINCDUX,Y,Z AND COSCDUX,Y,Z BY
039114,000091:                                                                                                  #                        WRITING INTO THESE LOCATIONS THE SINES AND COSINES
039115,000092:                                                                                                  #                        OF THE CDUD'S IN PNGCS-AUTO, OF THE CDU'S OTHERWISE.
