Source Code

These source-code files were transcribed from scans made from Don Eyles's personal copy of Luminary 069. They were scanned at archive.org's Boston facility, and the scanning was sponsored by Onno Hommes. The code was transcribed from these scans by a team of volunteers who are referenced in the program comments. Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". In some cases, where similar code blocks exist in previously-transcribed AGC programs (primarily Luminary 99, from Apollo 11) those code blocks were used as a starting point and then corrected to agree with the Luminary 69 scans. The full scans are available at the Virtual AGC project's collection at archive.org, while more-convenient reduced-size (but reduced-quality) images are available at the main Virtual AGC website. Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 069 OF AGC PROGRAM LUMINARY BY NASA 2021112-011
	19:02 NOV. 25,1968
Note that the date is the date of the printout, not the date of the program revision.

043536,000029:                                                                                                  #  PROGRAM NAME - DOWN TELEMETRY PROGRAM
043537,000030:                                                                                                  #  MOD NO. - 0           TO COMPLETELY REWRITE THE DOWN TELEMETRY PROGRAM AND DOWNLINK ERASABLE DUMP PROGRAM FOR THE
043538,000031:                                                                                                  #                        PURPOSE OF SAVING APPROXIMATELY 150 WORDS OF CORE STORAGE.
043539,000032:                                                                                                  #                        THIS CHANGE REQUIRES AN ENTIRELY NEW METHOD OF SPECIFYING DOWNLINK LISTS.REFER TO DOWNLINK
043540,000033:                                                                                                  #                        LISTS LOG SECTION FOR MORE DETAILS. HOWEVER THIS CHANGE WILL NOT AFFECT THE GROUND PROCESSING
043541,000034:                                                                                                  #                        OF DOWN TELEMETRY DATA.

043543,000036:                                                                                                  #  DATE - 02OCT67

043545,000038:                                                                                                  #  LOG SECTION - DOWN-TELEMETRY PROGRAM
043546,000039: 
043547,000040:                                                                                                  #  FUNCTIONAL DESCRIPTION - THIS ROUTINE IS INITIATED BY TELEMETRY END
043548,000041:                                                                                                  #        PULSE FROM THE DOWNLINK TELEMETRY CONVERTER. THIS PULSE OCCURS
043549,000042:                                                                                                  #        AT 50 TIMES PER SEC (EVERY 20 MS) THEREFORE DODOWNTM IS
043550,000043:                                                                                                  #        EXECUTED AT THESE RATES. THIS ROUTINE SELECTS THE APPROPRIATE
043551,000044:                                                                                                  #        AGC DATA TO BE TRANSMITTED DOWNLINK AND LOADS IT INTO OUTPUT
043552,000045:                                                                                                  #        CHANNELS 34 AND 35. THE INFORMATION IS THEN GATED OUT FROM THE
043553,000046:                                                                                                  #        LGC IN SERIAL FASHION.
043554,000047:                                                                                                  #        THIS PROGRAM IS CODED FOR A 2 SECOND DOWNLIST. SINCE DOWNRUPTS
043555,000048:                                                                                                  #        OCCUR EVERY 20 MS AND 2 AGC COMPUTER WORDS CAN BE PLACED IN
043556,000049:                                                                                                  #        CHANNELS 34 AND 35 DURING EACH DOWNRUPT THE PROGRAM IS CAPABLE
043557,000050:                                                                                                  #        OF SENDING 200 AGC WORDS EVERY 2 SECONDS.
043558,000051:                                                                                                  #  CALLING SEQUENCE - NONE
043559,000052:                                                                                                  #        PROGRAM IS ENTERED VIA TCF DODOWNTM WHICH IS EXECUTED AS A
043560,000053:                                                                                                  #        RESULT OF A DOWNRUPT. CONTROL IS RETURNED VIA TCF RESUME WHICH
043561,000054:                                                                                                  #        IN EFFECT IS A RESUME.
043562,000055:                                                                                                  #  SUBROUTINES CALLED - NONE
043563,000056:                                                                                                  #  NORMAL EXIT MODE - TCF RESUME
043564,000057:                                                                                                  #  ALARM OR ABORT EXIT MODE - NONE
043565,000058:                                                                                                  #  RESTART PROTECTION:
043566,000059:                                                                                                  #        ON A FRESH START AND RESTART THE 'STARTSUB' SUBROUTINE WILL INITIALIZE THE DOWNLIST POINTER (ACTUALLY
043567,000060:                                                                                                  #        DNTMGOTO) TO THE BEGINNING OF THE CURRENT DOWNLIST (I.E. CURRENT CONTENTS OF DNLSTADR). THIS HAS THE
043568,000061:                                                                                                  #        EFFECT OF IGNORING THE REMAINDER OF THE DOWNLIST WHICH THE DOWN-TELEMETRY PROGRAM WAS WORKING ON WHEN
043569,000062:                                                                                                  #        THE RESTART (OR FRESH START) OCCURRED AND RESUME DOWN TELEMETRY FROM THE BEGINNING OF THE CURRENT
043570,000063:                                                                                                  #        DOWNLIST.
043571,000064:                                                                                                  #        ALSO OF INTEREST IS THE FACT THAT ON A RESTART THE AGC WILL ZERO DOWNLINK CHANNELS 13, 34 AND 35.
043572,000065:                                                                                                  #  DOWNLINK LIST SELECTION:
043573,000066:                                                                                                  #        THE APPROPRIATE DOWNLINK LISTS ARE SELECTED BY THE FOLLOWING:
043574,000067:                                                                                                  #        1.      FRESH START
043575,000068:                                                                                                  #        2.      V37EXXE WHERE XX = THE MAJOR MODE BEING SELECTED.
043576,000069:                                                                                                  #        3.      UPDATE PROGRAM (P27)
043577,000070:                                                                                                  #        4.      NON-V37 SELECTABLE TYPE PROGRAMS (E.G. AGS INITIALIZATION (SUNDANCE, LUMINARY) AND P61-P62
043578,000071:                                                                                                  #                TRANSITION (COLOSSUS) ETC.).
043579,000072:                                                                                                  #  DOWNLINK LIST RULES AND LIMITATIONS:
043580,000073:                                                                                                  #        READ SECTION(S) WHICH FOLLOW 'DEBRIS' WRITEUP.
043581,000074:                                                                                                  #  OUTPUT - EVERY 2 SECONDS 100 DOUBLE PRECISION WORDS (I.E. 200 LGC
043582,000075:                                                                                                  #        COMPUTER WORDS) ARE TRANSMITTED VIA DOWNLINK.
043583,000076:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED - NONE
043584,000077:                                                                                                  #        'DNTMGOTO' AND 'DNLSTADR' ARE INITIALIZED BY THE FRESH START PROGRAM.
043585,000078:                                                                                                  #  DEBRIS (ERASABLE LOCATIONS DESTROYED BY THIS PROGRAM) -
043586,000079:                                                                                                  #        LDATALST,DNTMBUFF TO DNTMBUFF +21D,TMINDEX,DNQ.

043595,000088:                                                                                                  #  NOTES REGARDING DOWNLINK LISTS ASSOCIATED WITH THIS PROGRAM:
043596,000089:                                                                                                  #  1.    DOWNLISTS. - DOWNLISTS MUST BE COMPILED IN THE SAME BANK AS THE
043597,000090:                                                                                                  #        DOWN-TELEMETRY PROGRAM. THIS IS DONE FOR EASE OF CODING, FASTER
043598,000091:                                                                                                  #        EXECUTION.
043599,000092:                                                                                                  #  2.    EACH DOWNLINK LIST CONSISTS OF A CONTROL LIST AND A NUMBER OF
043600,000093:                                                                                                  #        SUBLISTS.
043601,000094:                                                                                                  #  3.    A SUBLIST REFERS TO A SNAPSHOT OR DATA COMMON TO THE SAME OR OTHER
043602,000095:                                                                                                  #        DOWNLINK LISTS.  ANY SUBLIST CONTAINING COMMON DATA NEEDS TO BE
043603,000096:                                                                                                  #        CODED ONLY ONCE FOR THE APPLICABLE DOWNLINK LISTS.
043604,000097:                                                                                                  #  4.    SNAPSHOT SUBLISTS REFER SPECIFICALLY TO HOMOGENOUS DATA WHICH MUST BE
043605,000098:                                                                                                  #        SAVED IN A BUFFER DURING ONE DOWNRUPT.
043606,000099:                                                                                                  #  5.    THE 1DNADR FOR THE 1ST WORD OF SNAPSHOT DATA IS FOUND AT THE END
043607,000100:                                                                                                  #        OF EACH SNAPSHOT SUBLIST, SINCE THE PROGRAM CODING SENDS THIS DP WORD
043608,000101:                                                                                                  #        IMMEDIATELY AFTER STORING THE OTHERS IN THE SNAPSHOT BUFFER.
043609,000102:                                                                                                  #  6.    ALL LISTS ARE COMBINATIONS OF CODED ERASABLE ADDRESS CONSTANTS
043610,000103:                                                                                                  #        CREATED FOR THE DOWNLIST PROGRAM.
043611,000104:                                                                                                  #        A.      1DNADR                  1-WORD DOWNLIST ADDRESS.
043612,000105:                                                                                                  #                SAME AS ECADR, BUT USED WHEN THE WORD ADDRESSED IS THE LEFT
043613,000106:                                                                                                  #                HALF OF A DOUBLE-PRECISION WORD FOR DOWN TELEMETRY.
043614,000107:                                                                                                  #        B.      2DNADR - 6DNADR         N-WORD DOWNLIST ADDRESS, N = 2 - 6.
043615,000108:                                                                                                  #                SAME AS 1DNADR, BUT WITH THE 4 UNUSED BITS OF THE ECADR FORMAT
043616,000109:                                                                                                  #                FILLED IN WITH 0001-0101.  USED TO POINT TO A LIST OF N DOUBLE-
043617,000110:                                                                                                  #                PRECISION WORDS, STORED CONSECUTIVELY, FOR DOWN TELEMETRY.
043618,000111:                                                                                                  #        C.      DNCHAN                  DOWNLIST CHANNEL ADDRESS.
043619,000112:                                                                                                  #                SAME AS 1DNADR, BUT WITH PREFIX BITS 0111.  USED TO POINT TO
043620,000113:                                                                                                  #                A PAIR OF CHANNELS FOR DOWN TELEMETRY.
043621,000114:                                                                                                  #        D.      DNPTR                   DOWN TELEMETRY SUBLIST POINTER.
043622,000115:                                                                                                  #                SAME AS CAF BUT TAGGED AS A CONSTANT. USED IN CONTROL LIST TO POINT TO A SUBLIST.
043623,000116:                                                                                                  #                CAUTION--- A DNPTR CANNOT BE USED IN A SUBLIST.
043624,000117:                                                                                                  #  7.    THE WORD ORDER CODE IS SET TO ZERO AT THE BEGINNING OF EACH DOWNLIST (I.E. CONTROL LIST) AND WHEN
043625,000118:                                                                                                  #        A '1DNADR TIME2' IS DETECTED IN THE CONTROL LIST (ONLY).
043626,000119:                                                                                                  #  8.    IN THE SNAPSHOT SUBLIST ONLY, THE DNADR'S CANNOT POINT TO THE FIRST WORD OF ANY EBANK.

043629,000122:                                                                                                  #    (THE FOLLOWING POINTS MAY BE LISTED ELSEWHERE BUT ARE LISTED HERE SO IT IS CLEAR THAT THESE THINGS CANNOT BE

043631,000124:                                                                                                  #  1.    SNAPSHOT DOWNLIST:

043752,000245:                                                                                                  #  THE FOLLOWING CODING (FROM SNAPLOOP TO SNAPEND) IS FOR THE PURPOSE OF TAKING A SNAPSHOT OF 12 DP REGISTERS.
043753,000246:                                                                                                  #  THIS IS DONE BY SAVING 11 DP REGISTERS IN DNTMBUFF AND SENDING THE FIRST DP WORD IMMEDIATELY.
043754,000247:                                                                                                  #  THE SNAPSHOT PROCESSING IS THE MOST TIME CONSUMING AND THEREFORE THE CODING AND LIST STRUCTURE WERE DESIGNED
043755,000248:                                                                                                  #  TO MINIMIZE TIME. THE TIME OPTIMIZATION RESULTS IN RULES UNIQUE TO THE SNAPSHOT PORTION OF THE DOWNLIST.

043832,000325:                                                                                                  #  SUBROUTINE NAME- DNDUMP
043833,000326:                                                                                                  #  FUNCTIONAL DESCRIPTION - TO SEND (DUMP) ALL ERASABLE STORAGE 'N' TIMES.(N=1 TO 4). BANKS ARE SENT ONE AT A TIME
043834,000327:                                                                                                  #        EACH BANK IS PRECEEDED BY AN ID WORD, SYNCH BITS, ECADR AND TIME1 FOLLOWED BY THE 256D WORDS OF EACH
043835,000328:                                                                                                  #        EBANK. EBANKS ARE DUMPED IN ORDER (I.E. EBANK 0 FIRST, THEN EBANK1 ETC.)
043836,000329:                                                                                                  #  CALLING SEQUENCE- THE GROUND OR ASTRONAUT BY KEYING V74E CAN INITIALIZE THE DUMP.
043837,000330:                                                                                                  #        AFTER KEYING IN V74E THE CURRENT DOWNLIST WILL BE IMMEDIATELY TERMINATED AND THE DOWNLINK ERASABLE DUMP
043838,000331:                                                                                                  #        WILL BEGIN.
043839,000332:                                                                                                  #        ONCE INITIATED THE DOWNLINK ERASABLE DUMP CAN BE TERMINATED (AND INTERRUPTED DOWNLIST REINSTATED) ONLY
