Source Code

These source-code files are part of a reconstructed copy of Luminary 163, the first (unflown) release of the Apollo 14 Lunar Module (LM) Apollo Guidance Computer (AGC) software.

The reconstruction began with reconstructed source code for Luminary 173. Changes between revision 163 and 173 were backed out, as described by Luminary memos 157 and 158. The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021152N. Note that page numbers in the reconstructed code match those for the baseline log section mentioned in each file's changelog; the page numbers for a real Luminary 163 listing would be different.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

058409,000167:                                                                                                  #  GENTRAN, A BLOCK TRANSFER ROUTINE.
058410,000168: 
058411,000169:                                                                                                  #  WRITTEN BY D. EYLES
058412,000170:                                                                                                  #  MOD 1 BY KERNAN                                                         UTILITYM REV 17 11/18/67
058413,000171: 
058414,000172:                                                                                                  #  MOD 2 BY SCHULENBERG  (REMOVE RELINT)   SKIPPER REV 4 2/28/68
058415,000173: 
058416,000174:                                                                                                  #           THIS ROUTINE IS USEFULL FOR TRANSFERING N CONSECUTIVE ERASABLE OR FIXED QUANTITIES TO SOME OTHER N
058417,000175:                                                                                                  #  CONSECUTIVE ERASABLE LOCATIONS.  IF BOTH BLOCKS OF DATA ARE IN SWITCHABLE EBANKS, THEY MUST BE IN THE SAME ONE.
058418,000176: 
058419,000177:                                                                                                  #           GENTRAN IS CALLABLE IN A JOB AS WELL AS A RUPT.  THE CALLING SEQUENCE IS:
058420,000178: 
058421,000179:                                                                                                  #                                          I        CA     N-1             # OF QUANTITIES MINUS ONE.
058422,000180:                                                                                                  #                                          I +1     TC     GENTRAN         IN FIXED-FIXED.
058423,000181:                                                                                                  #                                          I +2     ADRES  L               STARTING ADRES OF DATA TO BE MOVED.
058424,000182:                                                                                                  #                                          I +3     ADRES  M               STARTING ADRES OF DUPLICATION BLOCK.
058425,000183:                                                                                                  #                                          I +4                            RETURNS HERE.
058426,000184: 
058427,000185:                                                                                                  #           GENTRAN TAKES 25 MCT'S (300 MICROSECONDS) PER ITEM + 5 MCT'S (60 MICS) FOR ENTERING AND EXITING.
058428,000186: 
058429,000187:                                                                                                  #           A, L AND ITEMP1 ARE NOT PRESERVED.
058430,000188: 
058431,000189:    5555                                           BLOCK    02                                    
058432,000190:    4000                                           SETLOC   FFTAG4                                
058433,000191:    4000                                           BANK                                           
058434,000192: 
058435,000193:    5555     0061                                  EBANK=   ITEMP1                                
058436,000194: 
058437,000195:    5555                                           COUNT*   $$/TRAN                               
058438,000196: 
058439,000197:    5555           00004        GENTRAN            INHINT                                         
058440,000198:    5556           54061                           TS       ITEMP1                                #  SAVE N-1.

058445,000203:    5563           54001                           TS       L                                     #  SAVE DATA.
