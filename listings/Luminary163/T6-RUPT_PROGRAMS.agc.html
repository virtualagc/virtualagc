Source Code

These source-code files are part of a reconstructed copy of Luminary 163, the first (unflown) release of the Apollo 14 Lunar Module (LM) Apollo Guidance Computer (AGC) software.

The reconstruction began with reconstructed source code for Luminary 173. Changes between revision 163 and 173 were backed out, as described by Luminary memos 157 and 158. The reconstruction was verified by matching memory-bank checksums to those listed in drawing 2021152N. Note that page numbers in the reconstructed code match those for the baseline log section mentioned in each file's changelog; the page numbers for a real Luminary 163 listing would be different.

Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository.

059510,000021:                                                                                                  #  PROGRAM NAMES:  (1) T6JOBCHK      MOD. NO. 5    OCTOBER 2, 1967
059511,000022:                                                                                                  #                 (2) DOT6RUPT
059512,000023: 
059513,000024:                                                                                                  #  MODIFICATION BY: LOWELL G HULL (A.C.ELECTRONICS)
059514,000025:                                                                                                  #  THESE PROGRAMS ENABLE THE LM DAP TO CONTROL THE THRUST TIMES OF THE REACTION CONTROL SYSTEM JETS BY USING TIME6.
059515,000026:                                                                                                  #  SINCE THE LM DAP MAINTAINS EXCLUSIVE CONTROL OVER TIME6 AND ITS INTERRUPTS, THE FOLLOWING CONVENTIONS HAVE BEEN
059516,000027:                                                                                                  #  ESTABLISHED AND MUST NOT BE TAMPERED WITH:
059517,000028:                                                                                                  #           1. NO NUMBER IS EVER PLACED INTO TIME6 EXCEPT BY LM DAP.
059518,000029:                                                                                                  #           2. NO PROGRAM OTHER THAN LM DAP ENABLES THE TIME6 COUNTER.
059519,000030:                                                                                                  #           3. TO USE TIME6, THE FOLLOWING SEQUENCE IS ALWAYS EMPLOYED:
059520,000031:                                                                                                  #                  A. A POSITIVE (NON-ZERO) NUMBER IS STORED IN TIME6.
059521,000032:                                                                                                  #                  B. THE TIME6 CLOCK IS ENABLED.
059522,000033:                                                                                                  #                  C. TIME6 IS INTERROGATED AND IS:
059523,000034:                                                                                                  #                         I. NEVER FOUND NEGATIVE(NON-ZERO) OR +0.
059524,000035:                                                                                                  #                        II. SOMETIMES FOUND POSITIVE(BETWEEN 1 AND 240D) INDICATING THAT IT IS ACTIVE.
059525,000036:                                                                                                  #                       III. SOMETIMES FOUND POSMAX INDICATING THAT IT IS INACTIVE AND NOT ENABLED.
059526,000037:                                                                                                  #                        IV. SOMETIMES FOUND NEGATIVE ZERO INDICATING THAT:
059527,000038:                                                                                                  #                                A. A T6RUPT IS ABOUT TO OCCUR AT THE NEXT DINC, OR
059528,000039:                                                                                                  #                                B. A T6RUPT IS WAITING IN THE PRIORITY CHAIN, OR
059529,000040:                                                                                                  #                                C. A T6RUPT IS IN PROCESS NOW.
059530,000041:                                                                                                  #           4) ALL PROGRAMS WHICH OPERATE IN EITHER INTERRUPT MODE OR WITH INTERRUPT INHIBITED MUST CALL T6JOBCHK
059531,000042:                                                                                                  #              EVERY 5 MILLISECONDS TO PROCESS A POSSIBLE WAITING T6RUPT BEFORE IT CAN BE HONORED BY THE HARDWARE.
059532,000043: 
059533,000044:                                                                                                  #          (5. PROGRAM JTLST, IN Q,R-AXES, HANDLES THE INPUT LIST.)
059534,000045:                                                                                                  #  T6JOBCHK CALLING SEQUENCE:
059535,000046: 
059536,000047:                                                                                                  #                                          L        TC     T6JOBCHK
059537,000048:                                                                                                  #                                          L +1    (RETURN)
059538,000049: 
059539,000050:                                                                                                  #  DOT6RUPT CALLING SEQUENCE:
059540,000051: 
059541,000052:                                                                                                  #                                                   DXCH   ARUPT           T6RUPT LEAD IN AT LOCATION 4004.
059542,000053:                                                                                                  #                                                   EXTEND
059543,000054:                                                                                                  #                                                   DCA    T6ADR
059544,000055:                                                                                                  #                                                   DTCB
059545,000056: 
059546,000057:                                                                                                  #  SUBROUTINES CALLED: DOT6RUPT CALLS T6JOBCHK.
059547,000058: 
059548,000059:                                                                                                  #  NORMAL EXIT MODES: T6JOBCHK RETURNS TO L +1.
059549,000060:                                                                                                  #                     DOT6RUPT TRANSFERS CONTROL TO RESUME.
059550,000061: 
059551,000062:                                                                                                  #  ALARM/ABORT MODES: NONE.
059552,000063: 
059553,000064:                                                                                                  #  INPUT: TIME6    NXT6ADR      OUTPUT: TIME6     NXT6ADR     CHANNEL 5
059554,000065:                                                                                                  #         T6NEXT   T6NEXT +1            T6NEXT    T6NEXT +1   CHANNEL 6
059555,000066:                                                                                                  #         T6FURTHA T6FURTHA +1          T6FURTHA  T6FURTHA +1 BIT15/CH13
059556,000067:                                                                                                  #  DEBRIS: T6JOBCHK CLOBBERS A.  DOT6RUPT CLOBBERS NOTHING.
